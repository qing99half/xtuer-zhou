# 湘潭大学大学生艺术团

湘潭大学大学生艺术团（**校艺术团**）是校团委直接领导的 **校级官方文艺学生组织**，成立于 **1999 年**。汇聚全校热爱音乐、舞蹈、戏剧、舞台技术的学生，是校内 **规模最大、专业度最高** 的文艺团体。

## 一句话理解

- 校艺术团现有 **团员 500 余人**。
- 斩获 **国家级、省级文艺奖项百余项**。
- 曾赴 **澳门、西班牙** 开展文化交流演出。

## 一、组织定位

- **宗旨**：弘扬传统文化，繁荣校园文艺，提升学生审美素养，打造湘大特色美育平台。
- **核心职能**：承办全校大型晚会、代表学校参与省级 / 国家级文艺赛事、开展对外文化交流、完成红色文艺实践活动。

## 二、组织架构

### 主席团

- 设 **团长**、**团总支书记**、**4 名副团长**。
- 统筹全团训练、演出、人事、对外对接；分管各部门工作。

### 三大行政职能部门

1. **办公室**（行政中枢）
   - 档案管理、财务报销、会议记录、团员考勤、场地物资统筹、团内制度建设。
2. **外联部**
   - 对接商家获取活动赞助、管理演出服装道具、联动省内高校艺术团开展交流活动。
3. **舞台与设计中心（舞美宣传）**
   - 全校晚会 **灯光、音响、大屏** 技术保障。
   - 运营艺术团新媒体账号；晚会 **摄影、海报设计、视频剪辑**。
   - 管理学生活动中心报告厅舞台设备。

### 七大专业演职团队

1. **声乐部**：美声、流行合唱、独唱；晚会歌曲节目、合唱专场音乐会。
2. **舞蹈部**：民族舞、古典舞；代表作《花瑶女》曾赴澳门展演。
3. **VRF 街舞队**：爵士、breaking、popping；校园音乐节、军训巡演、街舞专场。
4. **民族乐团**：二胡、琵琶、古筝、扬琴；《浏阳河》《十面埋伏》等传统曲目。
5. **交响乐团**：管弦乐；举办中西交响专场音乐会。
6. **电声乐队**：吉他、贝斯、架子鼓；校园音乐节主力演出队伍。
7. **曲艺表演部**：小品、朗诵、情景剧、红色主题舞台剧。

## 三、常见问题

### Q1. 没有专业基础能进吗？

- 部分部门（如舞美宣传、外联、办公室）对专业基础要求较低。
- 演职团队通常需要一定基础，招新会有考核。

### Q2. 排练强度大吗？

- 大型晚会临近时排练密集，可能占用晚上或周末。
- 平时训练节奏相对可控。

### Q3. 有没有对外演出机会？

- 有。艺术团常代表学校参加省级、国家级赛事，也承担对外文化交流演出。

### Q4. 加入艺术团对保研 / 综测有帮助吗？

- 加入学生组织的工作经历通常可用于综测加分和奖学金参考，具体规则以本级本科生实施办法为准。

## 使用小建议

1. 想在大学阶段坚持艺术爱好、走上校级舞台的同学，艺术团是最佳平台之一。
2. 报到后关注 **校团委和艺术团公众号**，了解招新公告与选拔时间。
3. 招新准备：准备一段 **1–3 分钟才艺展示**（唱、跳、乐器等）。
4. 涉及各团队具体招新时间、考核细则，请以艺术团当年公告为准。
