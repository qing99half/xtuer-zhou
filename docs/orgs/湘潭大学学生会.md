# 湘潭大学学生会

湘潭大学学生会（校学生会）是学校党委、行政领导下成长的学生群众性组织，是 **中华全国学生联合会的基层组织**，也是学生进行 **自我教育、自我管理、自我服务、自我监督** 的组织和领导集体。

## 一句话理解

- 由 **共青团湘潭大学委员会** 受学校党委、行政委托进行管理和工作指导。
- 定位：**服务全体在校学生**，从班级事务到全校文体赛事全面覆盖。
- 秉承 “**团结、务实、和谐、创新**” 的工作作风。

## 核心职能

- 充分发挥 **学校与广大同学之间的桥梁纽带作用**。
- 积极动员、团结全校学生。
- 广泛开展各种生动活泼、健康有益的 **校园活动**。
- 创造良好的学习生活环境，为学生成长成才服务。

## 品牌活动

- 新生英语诵读大赛
- 中外文化交流节
- “百灵鸟” 杯辩论赛
- 青马工程卓越班
- 饮食文化节
- 志愿者文化节

上述活动通常成为全校师生关注的焦点。

## 官方入口

- 湘潭大学团委官网（校学生会介绍页）：`https://tw.xtu.edu.cn/qcfc/txzz.htm`

## 与其他学生组织的区别

- **校学生会**：全体在校本科生；聚焦班级事务、学风建设、权益维权、全校文体赛事、学生生活服务。
- **校研究生会**：仅面向硕博研究生。
- **校社团联合会**：管理和扶持全校 60 余个学生社团。
- **校大学生艺术团**：校团委领导的官方文艺组织，专业度更高。

## 新生加入常见问题

### Q1. 加入学生会有什么好处？

- 锻炼组织协调、活动策划、公文写作、新媒体运营等能力。
- 学生工作经历可用于 **综测加分、评优评先、奖学金** 参考。
- 结识更多同学、了解校内官方通知。

### Q2. 什么时候招新？

- 通常在开学初或百团大战期间集中招新，具体时间以校学生会公告为准。

### Q3. 招新流程？

- 现场 / 线上报名 → 面试 → 部门分配 → 正式成为干事。
- 部分部门有笔试或才艺展示要求。

## 使用小建议

1. 想深入了解学生工作、担任学生干部的同学可以尝试加入。
2. 想 “边玩边成长” 的同学更适合加入社团 / 艺术团 / 兴趣小组。
3. 涉及本年度招新细节，请以校团委、校学生会公告为准。
