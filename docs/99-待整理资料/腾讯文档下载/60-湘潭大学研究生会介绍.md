湘潭大学研究生会介绍
一、组织定位
湘潭大学研究生会是在校党委领导、研究生院与校团委双重指导下，面向全体硕士、博士研究生的校级官方学生组织，隶属中华全国学生联合会、湖南省学生联会。
核心宗旨：全心全意为研究生群体服务，搭建研究生与学校职能部门、导师之间的沟通桥梁，兼顾学术科研提升、生活权益保障、文体身心发展三大核心方向。
二、组织架构
主席团
设主席1名、副主席若干，统筹研会全年工作规划、部门管理、对外交流，对接研究生院、校团委，统筹大型品牌活动。
五大职能中心
1.
综合办公中心
研会行政中枢，负责会议组织、档案物资管理、财务报销、人员考勤、内部考核、通知传达，统筹各部门协同工作。
2.
学术发展中心
核心学术板块，承办全校研究生学术文化节、韶风名师讲坛、湘韵研究生论坛、硕博科研经验分享会、论文写作沙龙、学科竞赛交流活动，搭建跨专业科研交流平台。
3.
权益服务中心
专属研究生维权部门，收集实验室设备、宿舍后勤、奖助政策、食堂餐饮、科研报销等各类诉求，定期对接研究生院、后勤管理处开展座谈，落实同学合理诉求，发布权益科普推送。
4.
文体活动中心
策划研究生专属文体赛事与文艺活动：飞羽杯羽毛球赛、五四杯篮球赛、迎新晚会、毕业季文艺汇演、户外素质拓展、读书会等，丰富科研之外的课余生活。
5.
新媒体宣传中心
运营研会官方公众号、视频号，负责活动摄影摄像、推文撰写排版、海报设计、短视频制作，对外宣传研究生学术、文体、权益相关活动。
三、特色品牌活动
1.
学术类：研究生学术文化节、韶风名师论坛、硕博科研经验分享会、学术论文培训沙龙
2.
权益类：研究生权益座谈会、科研后勤意见征集、奖助学金政策科普
3.
文体类：研究生迎新晚会、毕业晚会、飞羽杯球类赛事、户外素质拓展
4.
思想建设类：研究生青马工程、红色研学实践活动
四、核心职能
1.
学术赋能：搭建科研交流平台，拓宽研究生学术视野，助力科研能力提升；
2.
权益保障：常态化收集、反馈、解决研究生学习、生活、科研相关问题；
3.
文化建设：打造适配研究生群体的文体活动，舒缓科研压力；
4.
桥梁纽带：传递学校政策，反馈学生诉求，参与校园研究生事务民主管理；
5.
思想引领：开展理论学习、红色实践活动，强化研究生思想建设。
五、与校学生会区别
1.
服务人群：研究生会仅面向全校硕博研究生；校学生会服务全体本科生；
2.
工作侧重：研究生会重点围绕科研、实验室、研究生奖助、毕业就业等研究生专属需求；校学生会聚焦本科班级、基础学风、基础校园生活服务。
六、加入价值
1.
锻炼公文写作、活动策划、沟通协调、新媒体运营等综合能力；
2.
学生工作经历可用于综测加分、评优评先、奖学金申报参考；
3.
跨专业结识硕博同学、对接校内名师，获取更多科研、学术活动资源；
4.
近距离对接研究生院、各学院老师，及时掌握奖助、答辩、就业等官方通知。
