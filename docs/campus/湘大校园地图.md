# 湘大校园地图

湘潭大学校园较大，第一次到校最容易在“教学楼—宿舍—食堂—图书馆”之间迷路。这份文字版校园地图先帮你把 **主要片区、常用地标、常用步行路径** 梳理清楚。

## 校园全景（先看这个）

- 校园 720 全景图：[720yun 校园全景](https://www.720yun.com/vr/ba2je5tOkn5)
  - 可以先在报到前用手机 / 电脑打开，转一圈，把宿舍、食堂、教学楼的空间位置装进脑子。
- 官方高清校园地图：请到学校官网 `www.xtu.edu.cn` “学校概况”或“生活设施”栏目获取。

## 主要片区（按方位理解）

湘大主要以 **兴湘校区（主校区）** 为核心，宿舍、食堂、教学楼、图书馆围绕琴湖、金翰林、南苑、北苑等生活区分布。

- **中心区**：图书馆、第一教学楼、南苑食堂群、南苑宿舍区。
- **北部**：北苑宿舍区、乐园食堂、北五食堂、经管楼、第三教学楼。
- **南部**：金翰林宿舍区、金翰林食堂、铜像广场、金翰林小吃街。
- **东南方向**：琴湖宿舍区、琴湖食堂、体育馆、琴湖水上景区。
- **兴湘学院内**：兴湘宿舍、兴湘餐厅、二环线附近。
- **兴湘片区（教学 / 科研）**：计算机中心与土木楼、环资院、法学学部、研究生院、信息科技大楼、自电院、数学院、材料院、分析测试中心、物理院、工科楼、信息实验楼、南山阶梯教室等聚集在兴湘片区。

## 常见地标（记住这些名字）

- **教学楼类**：第一教学楼、第三教学楼、南山阶梯教室、北山阶梯教室、逸夫楼、尚美楼、锻造楼、信息科技大楼、信息实验楼、南山阶梯教室、公共教学楼等。
- **办公 / 学院类**：外国语学院、公共管理学院、文学与新闻学院、机力院、商学院、碧泉书院、马克思主义学院、化工与化学学院、法学学部等。
- **公共服务类**：图书馆、教务处、学生活动中心、数学计算中心、分析测试中心。
- **食堂 / 生活类**：南苑师生餐厅、雅园厅、琴湖餐厅、金翰林餐厅、兴湘餐厅、乐园餐厅、北五餐厅、千喜鹤餐厅、云餐厅（爽提配送）。
- **地标景观**：铜像广场、琴湖、南山、北山。

## 常用步行路径（新生视角）

1. **宿舍 → 食堂**：绝大部分宿舍步行 5–10 分钟内至少有 2 个食堂可选。
2. **宿舍 → 教学楼**：从北苑 / 南苑到第一教学楼一般 10 分钟以内；从兴湘片区到主校区教学楼要留出 15–20 分钟。
3. **宿舍 → 图书馆**：图书馆位于中心区，从多数宿舍步行 10–15 分钟；期末周需要抢座建议提前 30 分钟出发。
4. **宿舍 → 体育馆**：琴湖公寓离体育馆最近；军训、体测集中场所以体育馆和田径场为主。

## 出行提醒

- 校区大，冬夏天温差大，跨片区上课建议自行安排电动车、共享单车（校内可用范围以校规为准）或统计课表按片区选宿舍。
- 校车 / 校内公交、片区间摆渡时刻表以后勤保障部或学校公众号最新通知为准。
- 迎新期间学校会安排志愿者引导，遇到路线不清可以直接问志愿者，比找地图更快。

## 使用小建议

1. 报到当天先弄清楚 **宿舍片区 + 最近食堂 + 最近教学楼** 三个坐标，其他地标可以后续摸索。
2. 手机地图先搜 “湘潭大学” 定位，再叠加高德 / 百度的“校内步行”路径，比看纸质地图快。
3. 高清地图 / 迎新手册通常会在开学前后发放，建议留存电子版备用。
4. 涉及具体的“某教学楼有哪些教室、几楼有饮水机、几栋在维修”，一律以校内公告、学院通知为准。
