# 信息门户使用指南

湘潭大学信息门户是智慧校园重要系统平台之一，是师生 **在线浏览信息、工作协作的统一入口**。这份指南讲清楚 “**登录方式 + 找回密码 + 常见问题**”。

## 一句话理解

- **入口**：`https://portal2020.xtu.edu.cn` 或从学校主页 `www.xtu.edu.cn` “信息门户” 链接进入。
- **账号**：学号（学生）/ 工号（教职工）。
- **初始密码**：身份证后六位（如末位为 X 请用大写）。
- **登录方式**：账号密码登录 / 湘大校园 App 扫码登录。

## 一、信息门户能干什么

登录信息门户后，可以通过 **一次认证** 进入多个业务系统：

- 教务系统
- 研究生系统
- 实验室综合管理平台
- 图书馆管理系统
- 学工系统
- 心理健康教育平台
- 财务智能平台
- 一卡通系统
- 湘大邮箱
- ……等几十个应用系统

## 二、登录方法

### 方式 A：用户名密码登录

- **用户名**：学号。
- **登录密码**：初始为身份证后六位（末位 X 大写）。
- 第一次登录会提醒添加 **密保手机**，用于找回密码。

### 方式 B：二维码扫码登录（推荐）

- 页面右侧为扫码登录区。
- 使用 **湘大校园 App** 的 “扫一扫” 功能扫码登录。
- 需要先在手机上下载 **湘大校园 App**，并完成认证和登录（参考 “新生服务” 中的 “湘大校园 App 下载及身份认证须知”）。
- 优点：**避免输错密码导致账号锁定**。

## 三、忘记密码找回

1. 在登录页面点击密码栏后的 **“忘记登录密码”** 链接，进入 “找回密码” 页面。
2. 按提示输入 **学号、身份证号及验证码**，点击下一步。
3. 选择 **“通过密保手机”** 找回密码。
4. 输入短信验证码，完成新密码设置即可。

## 四、常见问题

### Q1. 忘记密码但也没绑定密保手机怎么办？

- 拨打 **信息门户服务电话 0731-58292315** 咨询。
- 或者带身份证到 **南苑网信中心服务窗口** 现场处理。

### Q2. 密码输错锁定了？

- 一般等一段时间会自动解锁。
- 也可用 **扫码登录** 方式（不占用密码尝试次数）。

### Q3. 校外能不能用信息门户？

- 可以。信息门户本身在公网可访问，但里面某些系统仍需 **VPN**。
- VPN 使用见 [VPN 使用指南](/library/tools/VPN使用指南)。

### Q4. 密码复杂度要求？

- 一般要求 **8 位以上、大小写 + 数字 + 特殊字符**，每半年必须更换。

## 五、服务电话

- 信息门户服务电话：**0731-58292315**

## 使用小建议

1. 报到当天就登录一次信息门户，把 **密保手机** 绑定好，避免以后忘记密码求助无门。
2. 使用扫码登录能有效防止密码泄露和账号锁定。
3. 一次登录信息门户，会同时授权多个业务系统，不用每个系统单独登录。
4. 涉及具体操作截图和最新入口，请以学校信息化建设与管理处最新公告为准。
