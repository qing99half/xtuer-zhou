# 转专业指南

湘大转专业的整体思路是：**校定框架 + 学院细则**。学校发布总的管理办法（湘大教发〔2024〕2 号），具体的转出比例、转入考核由各学院自行制定。本科从 2023 级起按新办法执行。

## 一句话理解

- **转出**：不是完全无限制，各学院会给转出比例，例如物理与光电工程学院上限约 20%。
- **转入**：几乎所有学院都需要考核（笔试 + 面试 / 机试 + 专业应知应会评估等）。
- **时间**：多数学院集中在 **第一学期末** 或 **第三学期末**（大一或大二上开学第一周）。

## 不允许转专业的情形

有下列情形之一的学生，不允许转专业：

- 入学未满一学期。
- 本科三年级以上（含三年级）。
- 国家或学校在招生时明确规定不可转专业的（含中外合作办学、定向生、第二学士学位等）。
- 学校认定为不宜转专业的其他情形。

## 通用申请流程

具体流程以本学期通知为准，常见步骤：

1. **准备材料**（原学院）
   - 《湘潭大学学生转专业申请表》
   - 目标学院要求的补充材料（如《湘潭大学 XX 学院申请转专业审批表》）
   - 加盖原学院公章的必修 / 选修课程成绩单
   - 学生证复印件
   - 获奖证书、竞赛证明等（如目标学院认可）
2. **原学院审批**：在规定时间内递交至原学院学工办 / 教务办；原学院于预备周或第一周内完成转出审批。
3. **提交给目标学院**：按目标学院通知提交材料，参加考核。
4. **目标学院考核**：常见形式包括笔试、面试、机试、专业应知应会评估。
5. **公示 3 个工作日**：考核通过名单在目标学院网站公示。
6. **报送教务处**：公示无异议后由学院汇总报教务处，教务处审核后提交学校转专业工作领导小组审批。
7. **学籍变动**：由教务处完成新专业学籍变动及电子注册上报。

## 学院差异（举例说明）

### 计算机学院 · 网络空间安全学院

- **申请时间**：第一学期末或第三学期末课程结束后可申请。
- **前置条件**：上学期所有必修和选修课程无挂科。
- **考核方式**：C 语言程序设计机试 + 专业应知应会评估 + 面试。
- **机试不达标**：可以出具保证书参加面试，承诺当学期期末通过考核；如未通过则退回原专业。
- **学分不足**：根据转入专业培养计划审核，缺修 15 及以上课程学分数的会编班入低一年级。

### 机械工程与力学学院（2025-2026-1 学期实操示例）

- **提交时间**：2025-2026-1 学期开学第一周周三下午 17:00 前。
- **提交地点**：金瀚林 8 栋 304 办公室（转出）；转入学院地点见目标学院通知。
- **考核**：学院组织综合考核，转专业工作小组根据情况确定名单。
- **逾期**：未在规定时间内报名的不予受理。

### 数学与计算科学学院

- **笔试科目**：数学综合。
- **第二学期申请**：内容包括高等数学、数学分析、高等代数。
- **第三学期申请**：内容包括高等数学、线性代数。
- **附加要求**：学生须按学院要求提交申请表、佐证材料，对材料真实性负责；逾期或不服从工作安排的不予接收。

### 物理与光电工程学院

- **转出名额**：各专业申请转出人数不超过该专业相应年级入学注册人数的 20%。
- **转入名额**：各专业转入接收人数不超过该专业相应年级入学注册人数的 30%。
- **学院内转专业**：人数不计入转出 / 转入名额。
- **考核**：学院成立专家考评小组，综合考评并确定通过名单，公示 3 个工作日。

## 常见误区

- ❌ “转出完全无限制”：学校总办法交由各学院制定细则，多数学院有转出比例上限。
- ❌ “入学第一学期就能转”：不允许入学未满一学期就申请转专业。
- ❌ “大三还能转”：本科三年级（含三年级）以上不能申请转专业。
- ❌ “有挂科也能转”：多数学院要求 **上学期无挂科** 或 **必修课无不及格**。
- ❌ “口头承诺就行”：所有申请材料必须书面提交，逾期不受理。

## 时间节点建议

1. **入学第一学期**：了解自己专业和目标专业的培养方案差异，确认是否有转专业意愿。
2. **第一学期期末前**：留意 **原学院学工办、目标学院官网** 的转专业通知。
3. **第一学期末 / 第三学期初**：按目标学院时间节点提交申请材料。
4. **考核前**：针对目标学院要求的科目（数学综合、C 语言等）集中复习。
5. **公示期**：留意目标学院官网，如有异议要在公示期内实名反映。

## 使用小建议

1. 转专业是 “**校定框架 + 学院细则**”，一定要看目标学院当年官网通知，不要只看学校总办法。
2. 转专业成功后你会切换到新专业培养方案，之前修的课按新方案换算是否可抵学分。
3. 想提高成功率，除了成绩，也要在申请表和面试中说清楚 **转专业的动机、目标和长期规划**。
4. 如涉及具体学院、当年名额、笔试面试真题、往届经验，请以本校学长学姐、辅导员和目标学院教务办获取一手信息。
