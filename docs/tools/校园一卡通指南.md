# 校园一卡通指南

湘大 **校园卡** 由学校与中国建设银行湖南省分行合作发行，是 **实名制金融 IC 卡**，兼具建行龙卡 + 湘潭大学校内管理卡的双重功能。这份指南把 “**能干什么、怎么充值、怎么维护**” 讲清楚。

## 一句话理解

- 校园卡表面印有 **姓名、学（工）号和照片**，校内可代替 **工作证、学生证、退休证和考试证**。
- 卡内有 **两个账户**：**建行储蓄账户**（银行卡功能）+ **电子钱包账户**（校内消费）。
- 校园卡管理中心地址：**南苑建设银行北 150 米处**。

## 一、校园卡能干什么

### 校园身份识别

- **图书借阅、考勤、会议签到、上机、门禁、校医院体检、体能测试** 等。

### 两个账户

#### 1. 建行储蓄账户

- 具有 **银行卡的全部功能**。
- 可签约进行：
  - 校园网络缴费
  - 选修课、重修课、英语四六级报名
  - 学杂费缴费
  - 奖学金、助学金发放

#### 2. 电子钱包账户

- 用于 **就餐、购物、购电、交网费、宿舍热水、直饮水** 等校园刷卡消费。

## 二、充值方式

### 网络充值（推荐）

- **湘大校园 App**
- **支付宝**
- **中国建设银行 App**

> 网络充值的金额会先进入用户账号，在食堂 POS 机消费时系统自动将充值金额写入校园卡账户。

### 现金充值

- 到 **校园卡服务中心** 现金充值。

## 三、消费方式

1. 商户输入消费金额。
2. 将校园卡贴近读卡器。
3. 听到 **“嘀”** 声，消费金额从卡内扣除，屏幕显示余额。
4. 移开卡片，完成消费。

## 四、卡片养护

校园卡是 **金融 IC 卡**，卡片的夹层中含有天线和集成电路。

**注意事项**：

- 不要 **弯曲、打孔、划伤、水浸、油浸、重压**。
- 避免 **静电**。
- 卡片任何一部分损坏都可能影响正常使用。
- 建议放在 **硬质证件夹** 内保护。

## 五、丢失、拾获处理

- **拾获他人校园卡**：请及时交到校园卡服务中心，中心会通知用户认领。
- **盗用他人校园卡**：属于违法行为，将受到相关法律制裁。

## 七、常见问题

### Q1. 校园卡丢了怎么办？

- **挂失**：登录校园卡服务网址或到管理中心办理挂失。
- **补办**：本人携带身份证到 **校园卡管理中心** 补办。

### Q2. 校园卡余额不够怎么办？

- 用 **湘大校园 / 支付宝 / 建行 App** 充值即可。
- 也可现金到管理中心充值。

### Q3. 校园卡可以刷公交吗？

- 校园卡属校内消费；校外公交需要使用 **湘潭出行 App** 或电子公交卡。

### Q4. 忘记密码怎么找回？

- 拨打 **0731-58298578** 咨询，或到管理中心办理。

## 使用小建议

1. 报到后第一时间检查 **校园卡是否可用、余额是否到账**。
2. 平时使用建议 **多点小额充值**，避免一次充值过多丢卡损失。
3. 不要把校园卡和 IC 门禁卡放同一硬壳内以免相互干扰。
4. 涉及具体充值上限、消费限额，请以校园卡管理中心公告为准。
