# 故障申报与受理

湘大宿舍、教学楼、网络出故障时，主流报修入口是 **“湘大校园” App**（在线报修）+ **网信中心 / 后勤保障部电话**。这份指南把 “**在线报修、电话报修、宿舍报修**” 三种场景讲清楚。

## 一句话理解

- 在线报修：**“湘大校园” App → 生活应用 → 在线报修**。
- 电话报修：网信中心 **58293315 / 58298997**。
- 宿舍报修：走 **“湘大校园” App**，选择维修类别、报修区域和地址。

## 一、校园网报修流程

### 方式 A：湘大校园 App 报修

1. 手机登录 **“湘大校园” App**。
2. 点击 **“生活应用”** → 选择 **“在线报修”** → 进入后勤维修系统。
3. 点击 **“我要报修”**，按实际情况填写：
   - 维修类别
   - 报修区域
   - 报修地址
   - 故障描述
4. 确认无误后点击 **“提交”** 完成报修。
5. 点击 **“我的报修”** 查看故障处理情况。

### 特殊地址填写

- **金翰林公寓、兴湘 1–5 栋、北苑 3–8 栋** 等未在 “报修地址” 中罗列的地址，报修时选择 **“其他”**，并在 **“地址描述”** 里详细描述。

### 方式 B：电话报修

- **校园网故障报修电话**：**58293315、58298997**

## 二、宿舍报修

宿舍报修按同一流程走 “湘大校园” App：

### 第一步：下载 “湘大校园” App

- 账号：学号。
- 初始密码：**身份证号后六位**，字母需大写。

### 第二步：进入主页

- 选择 **“在线报修”**。

### 第三步：进入报修页面

- 点击 **“我要报修”**。

### 第四步：填写报修信息

- 维修类别（如水管、电器、门锁、家具等）。
- 详细描述故障。
- 联系电话保持畅通。

### 第五步：查看进度

- 点击 **“我的报修”** 查看处理进度。

## 三、按故障类型选入口

| 故障类型 | 报修入口 |
| --- | --- |
| 校园网络（宿舍宽带、WiFi） | 湘大校园 App 在线报修 / 电话 58293315 |
| 宿舍水电（灯管、水管、跳闸） | 湘大校园 App 在线报修（后勤维修系统） |
| 宿舍门锁 / 家具 | 湘大校园 App 在线报修 |
| 空调故障 | 湘大校园 App 在线报修（可能属后勤） |
| 教学楼公共设施故障 | 走教务办 / 后勤保障部 |
| 一卡通 / 校园卡问题 | 校园卡管理中心 58298578 |
| 缴费问题 | 计划财务处 / 湘大财务公众号 |

## 四、常见问题

### Q1. 报修提交后没有回应怎么办？

- 先在 **“我的报修”** 查看进度。
- 时间过长可通过电话或到 **南苑网络与信息中心** 现场登记。

### Q2. 宿舍紧急故障（漏水、跳闸）怎么办？

- 打宿舍片区物业电话或后勤紧急电话。
- 同时在 “湘大校园” App 报修留下记录。

### Q3. App 打不开怎么办？

- 检查 App 是否是最新版本。
- 尝试重新登录（账号 = 学号，初始密码为身份证后六位）。
- 通过电话报修作为备用。

### Q4. 空调不制冷是不是要自己找修？

- 湘大宿舍空调通常由学校统一管理，走 **在线报修 / 后勤保障部** 处理，不建议私自维修。

## 五、日常维护小习惯

1. 报到后就 **下载并登录 “湘大校园” App**，报修 / 缴费一站式搞定。
2. 常用故障电话可以 **保存在通讯录**（网信中心、后勤、校园卡中心）。
3. 遇到设施故障 **及时报修**，不要拖延，避免影响正常学习生活。
4. 涉及具体维修联系人、宿舍片区物业电话，请以宿舍公告为准。
