# 奖学金与助学金政策

湘潭大学的奖助学金体系分为 **国家资助、地方政府资助、校内资助** 三大类，覆盖本科生和研究生。这份指南先按 “**入学前 → 入学时 → 入学后**” 三个时间段梳理常见资助，再讲流程和材料，方便新生对号入座。

## 一句话理解

- 奖学金是 **奖励优秀**，助学金是 **保障家庭经济困难学生的基本生活**。
- 无论奖学金还是助学金，都要经过 **申请 → 学院评审 → 学校审核 → 公示** 的完整流程。
- 具体金额、名额、比例每年会变，通知发布通常集中在 **秋季学期开学后、评奖年度截止前**。

## 一、入学前可享受的资助

### 生源地信用助学贷款

- 湘大目前 **只开展生源地信用助学贷款**（不设校园地国家助学贷款）。
- **申请渠道**：入学前向户籍所在县 / 市 / 区的学生资助管理机构提出申请。
- **贷款额度**：本科生最高 **16000 元 / 年**。
- **用途**：优先支付学费和住宿费，超出部分用于日常生活费。
- **还款**：学生在校就读期间无需还款付息；最长贷款期限为 **剩余学制加 15 年、最长不超过 22 年**。

## 二、入学时可享受的资助

### 绿色通道

- 家庭经济特别困难的新生如暂时筹集不齐学费和住宿费，可在报到时通过 **绿色通道** 先办理入学手续。
- 入学后，校资助部门会开展家庭经济困难认定，按情况给予资助。
- 已办理生源地信用助学贷款的同学 **直接通过绿色通道入学**，不需要再向学校缴纳助学贷款部分的学费。

## 三、入学后可享受的资助（国家资助）

### 1. 国家奖学金

- 面向：全日制 **二年级以上（含二年级）** 本科生中特别优秀者。
- 金额：**8000 元 / 生 / 年**。
- 名额参考：湘大近年奖励约 **38 名**。
- 附加：颁发国家统一印制的获奖证书。

### 2. 国家励志奖学金

- 面向：全日制 **二年级以上（含二年级）** 品学兼优的家庭经济困难本科生。
- 金额：**5000 元 / 生 / 年**。
- 名额参考：湘大近年奖励约 **927 名**。

### 3. 国家助学金

- 面向：全日制本科生（含第二学士学位）中家庭经济困难的学生。
- 金额：平均资助标准 **3300 元 / 生 / 年**，分三档：
  - **一等 4400 元 / 二等 3300 元 / 三等 2200 元**。
- **在校退役士兵学生** 全部享受本科生国家助学金，标准 3300 元 / 生 / 年。

### 4. 学费补偿与贷款代偿

- **应征入伍服义务兵役 / 招收为军士 / 退役后复学或入学的学生**：享受学费补偿、国家助学贷款代偿、学费减免；标准按学校实际收取学费金额执行，本科生每生每年 **最高不超过 16000 元**。
- **基层就业学费补偿**：湖南省属全日制普通高校应届毕业生毕业后到基层单位就业，本科生 **5000 元 / 年**（第二学士学位 8000 元 / 年），连续补偿 **3 年**。

## 四、入学后可享受的资助（学校资助）

### 1. 学年综合奖学金

- 面向：全日制二年级以上（含二年级）道德品质优良、学业成绩优异的本科生。
- **甲等 1500 元 / 乙等 1000 元 / 丙等 500 元**。

### 2. 单项奖学金

- 拔尖人才培养奖助学金。
- 基础学科专业奖学金。
- 学校励志奖学金。
- 创新奖。
- 竞赛奖：
  - 学科竞赛奖。
  - 文艺、体育竞赛奖。
  - 集体奖。
- 国防奖学金。
- 其他单项奖。

> 学年综合奖学金、单项奖学金评选条件及要求参见《湘潭大学本科生奖学金实施办法》（湘大学发〔2022〕2 号）。

### 3. 特困生困难补助

- 学生在校期间出现 **突发性困难、家庭遭遇严重自然灾害、家庭发生重大变故** 等情况的可申请临时困难补助。
- 补助金额根据学生实际情况确定。

### 4. 勤工助学

- 学有余力的同学可通过 **湘大勤管中心兼职群** 了解兼职信息。
- 原则上时薪不低于 **12 元 / 小时**。

### 5. 其他资助

- 学校设有 **新生寒衣发放、春节慰问** 等暖心资助项目。

## 五、社会资助

- **京东奖助学金**
- **金桂助学金**
- **“伟人之托” 奖助学金**（湘大特色）
- **宝钢教育奖**

评选条件与资助标准依据资助方与学校签订的协议执行。

## 六、研究生奖学金 / 助学金

- **研究生国家奖学金**：面向研究生。
- **研究生学业奖学金**：分为硕士研究生学业奖学金、新生学业奖学金等，硕士学业奖学金最高 **10000 元**。
- **研究生国家助学金**：博士 **13000 元 / 生 / 年**，硕士 **6000 元 / 生 / 年**，基本实现应助尽助。
- 研究生任何时候终止学业的，从下个月起不再享受，并以每学年 **10 个月** 时间计算；如果已经预发，需退还。

## 七、通用申请流程

具体流程以学院和当年通知为准，常见步骤：

1. **学院通知**：辅导员或学工办发布评奖评助通知，明确时间节点、申请材料、评审细则。
2. **学生申请**：填写申请表，准备成绩单、家庭情况证明（针对助学金）、证书证明（针对奖学金）等。
3. **班级 / 年级评议**：部分项目需要班级或年级评议小组打分或投票。
4. **学院评审**：学院评审委员会汇总意见，形成推荐名单。
5. **学校审核**：由学校学生资助管理中心（本科）或研究生院（研究生）审核。
6. **公示**：公示期一般 5–10 天，接受师生实名反映意见。
7. **发放**：公示无异议后按学校财务时间表发放。

## 八、需要准备的常见材料

- **奖学金**：成绩单、绩点排名证明、获奖证书、发表论文证明、社会实践证明等。
- **助学金**：家庭经济困难认定材料（民政部门开具或按学校要求填写的困难认定表）、家庭成员信息证明等。
- **特殊项目**：按设奖方通知准备（如社会奖助学金、专项奖）。

## 九、咨询与申报入口

- **本科生**：向所在学院 **学生工作办公室 / 辅导员** 咨询。
- **研究生**：向所在学院 **研究生工作办公室** 咨询。
- **学校层面**：湘潭大学 **计划财务处**、**学生资助管理中心**（学生工作部下设）。
- 计划财务处入口：`https://cwc.xtu.edu.cn/`

## 十、常见误区

- ❌ “奖学金人人可申请”：部分校内奖学金对绩点排名、参与活动、社会工作有硬性门槛。
- ❌ “国家奖学金和国家励志奖学金能同时拿”：绝大多数情况下 **同一学年两者不能同时获得**，具体以当年通知为准。
- ❌ “助学金一定按学期发”：不同项目发放节奏不同，请按学校财务通知为准。
- ❌ “只有学习成绩重要”：许多校内奖学金综合考虑综合素质、社会工作、志愿服务、竞赛等。

## 十一、使用小建议

1. 大一进校第一个月就把辅导员、学院学工办的通知渠道加进来（微信、QQ 群、学院官网、公众号），不要错过评奖时间。
2. 提前准备好电子版和纸质版的证书、成绩单、家庭情况证明，评奖季不用临时翻找。
3. 家庭经济困难认定要 **真实、按规定流程办理**，不要造假；被查实会被取消资助资格并可能影响综合评价。
4. 关注 **“伟人之托” 奖助学金** 等湘大特色项目，容易被忽略。
5. 涉及金额、名额、比例的具体数字，一律以当年学校通知为准。
