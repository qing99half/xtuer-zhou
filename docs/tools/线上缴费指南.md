# 线上缴费指南

湘大的线上缴费主要涉及 **学费 / 学杂费** 和 **生活缴费**（水电、宽带等）。这份指南把两条路径、票据获取方式讲清楚。

## 一句话理解

- 学费缴费走 **“湘大财务” 微信公众号** 或 **“湘大校园” App**。
- 生活水电走 **“智慧湘大服务号”** 微信公众号。
- 从 2024 年起，全面启用 **电子凭证**，无法提供纸质票据。

## 一、学费 / 学杂费缴费路径

### 路径 1：微信公众号 “湘大财务” 缴费

- 关注 **“湘大财务”** 微信公众号。
- 按公众号推送或菜单进入缴费界面。
- 参照公众号内《本部学生微信缴费指引》操作。

### 路径 2：湘大校园 App

- 支持 **生源地助学贷款** 和 **银行转账** 方式缴费。
- 缴费路径无变化。

## 二、票据获取

### 电子缴款书替代纸质票

- 从 2024 年起，**全面启用电子凭证**。
- 以往的 “湖南省非税收入专用收据（学杂费电脑票）” 和 “湖南省非税收入电子缴款收据” 均由 **“电子缴款书”** 替代。
- 无论通过 **“湘大财务”** 还是 **“湘大校园” App** 缴费，均可在 **“湘大财务”** 中下载电子缴款书。
- 具体操作见公众号推送的《本部学生微信缴费指引》。

### 获取时间

- **微信公众号 “湘大财务” 缴费**：成功后 **10 分钟内** 可获取电子缴款书。
- **生源地助学贷款 / 银行转账缴费**：成功后 **2 个月左右** 可下载电子缴款书。

## 三、生活缴费（水电）

按 **“湘潭大学研究生”** 微信公众号公示的流程：

1. **第一步**：关注 **“智慧湘大服务号”**。
2. **第二步**：点击 **“后勤服务”** → 选择 **“水电缴费”**。
3. **第三步**：选择 **“账号绑定”**，注意核对 **区域 → 楼栋 → 房间号**。
4. **第四步**：选择 **“支付水电”**。

## 四、常见问题

### Q1. 交完学费能马上下载电子缴款书吗？

- 走 “湘大财务” 微信缴费的，10 分钟内即可。
- 走贷款 / 银行转账的，需要等 2 个月左右。

### Q2. 忘记绑定房间号，缴费缴错宿舍怎么办？

- 生活缴费严格按 **区域 → 楼栋 → 房间号** 绑定；如缴错请联系后勤保障部处理。

### Q3. 找不到 “湘大财务” 公众号怎么办？

- 在微信搜索栏输入 **“湘大财务”**，认准 **官方认证** 账号。
- 或到 **计划财务处** 官网 `https://cwc.xtu.edu.cn/` 查看官方入口。

### Q4. 学费缴费失败怎么办？

- 优先检查是否绑定正确的缴费卡、余额是否充足。
- 无法解决时联系 **计划财务处 0731-58293628**（以官方最新公告为准）。

## 五、缴费种类快速对照

| 缴费类型 | 常用入口 |
| --- | --- |
| 学费、住宿费、代收费 | 湘大财务微信公众号 / 湘大校园 App |
| 水电 | 智慧湘大服务号 |
| 校园宽带 | 电信 / 移动 / 联通 App 或校内营业厅 |
| 校园卡消费 | 湘大校园 App / 支付宝 / 建行 App |
| 网络充值 | 智慧湘大 / 校园卡管理中心网址 |

## 使用小建议

1. 缴费前 **核对金额**、缴费用途，避免误缴。
2. **电子缴款书** 下载后可保存到云端，避免二次找票。
3. 生活缴费时看清 **区域 + 楼栋 + 房间号**，缴错很难退款。
4. 涉及具体缴费入口的界面截图，请以本级官方公众号最新版本为准。
