# 校园政策 Q&A

这份 Q&A 覆盖 **学业学籍、考勤学风、宿舍管理、违纪处分、请假休学** 等新生高频关心的纪律与规则，帮你把常见红线一次记住。

## 一、学业学籍极简规则

1. 本科 **4 年制**，最长可读 **6 年**，超期注销学籍。
2. 挂科有 **补考**，补考不过需 **重修**；挂科记录永久留存；挂科当年通常取消所有奖助、评优、保研资格。
3. 转专业仅一次机会，大一下初申请，需 **无挂科、无违纪**，看绩点排名；艺术类、专项计划不可跨类转专业。
4. **作弊、严重学术不端、留校察看处分未解除，不授予学位**。

## 二、考勤学风规则

1. 学校查课严格，旷课累计达标 **直接取消课程考试资格、挂科**。
2. **代签到、虚假签到** 一律违纪通报，扣综测分。
3. 图书馆、自习室禁止 **占座、外放、喧哗、就餐**。

## 三、宿舍管理红线（最容易违规）

1. 禁止 **私调床位、私换寝室、留宿外人**，禁止 **私自校外住宿**。
2. 违禁必查必没收：**电煮锅、热得快、大功率电器、明火、管制刀具、改装插排**。
3. 有 **门禁**，晚归、夜不归宿会登记通报，多次违纪会给处分。
4. 宿舍卫生多次不合格，**全员** 取消当年评奖、助学金资格。

## 四、违纪处分极简说明

1. 处分等级：**通报批评 → 警告 → 严重警告 → 记过 → 留校察看 → 开除学籍**。
2. **考试作弊、打架斗殴、屡犯宿舍违规、学术造假** 直接重处分。
3. 所有处分入档案，处分期内：**不能评优、拿奖助、入党、保研、考选调**。

## 五、请假、休学规则

1. 上课、活动必须提前正规请假，**私自缺席按旷课处理**。
2. **因病、创业等** 可申请休学，保留学籍，复学按学校流程办理。

## 六、新生必守底线（一句话总结）

- **不挂科、不作弊、不违规用电器、不晚归外宿、不违纪**，即可保住所有 **评优、奖助、保研、毕业** 资格。

## 七、常见问题延展

### Q1. 挂了一门课，还能保研 / 拿奖学金吗？

- 挂科当年通常取消当年的评奖、评优和保研资格；具体到长期影响以推免细则和奖学金评审细则为准。
- 详见 [保研与科学竞赛](/library/study/保研与科学竞赛) 和 [奖学金与助学金政策](/library/tools/奖学金与助学金政策)。

### Q2. 违纪一次会不会影响毕业？

- 通报批评、警告等轻处分通常不会影响毕业，但会进档案影响评优。
- 留校察看、开除等重处分会直接影响学位授予或学籍。

### Q3. 请假找谁审批？

- 短期请假：辅导员。
- 长期 / 特殊情况：辅导员报学院学工办 / 教务办。
- 详见 [校园常见办事流程](/library/tools/办事流程总览)。

### Q4. 私自校外住宿被查会怎样？

- 属于违纪，会通报、扣综测分；多次违规会进入违纪档案。
- 涉及安全事故的还可能承担更严重责任。

### Q5. 我在网上看到 “湘大挂了就退学” 是真的吗？

- 不是。挂科可以补考、重修，一次挂科不会直接退学。
- 但连续挂科、长期学业预警未干预、超期未毕业，最终可能被 **注销学籍**。

## 使用小建议

1. 报到第一周就把这份 Q&A 通读一遍，避免因为 “不知道规则” 走弯路。
2. 涉及具体处分依据、评奖细则、补考规则，请以教务处、学生工作部当年发布的文件为准。
3. 有疑问不要在群里传谣，先问辅导员。
