# 选课与通识课

选课是大学生活最核心的操作之一。湘大的选课体系包括 **必修课**、**专业选修课** 和 **通识课 / 自主发展课程**。这份指南把三类课程分别讲清楚，帮你少踩坑。

## 一句话理解

- **必修课**：教务系统一般自动排入课表，不需要抢。
- **专业选修课**：在培养方案里预设，按学期选课窗口选择。
- **通识课 / 自主发展课程**：跨专业跨学院开放，选课高峰几分钟内就可能被抢完，动作要快。

## 教务系统入口

- 湘大教务处官网：`https://jwc.xtu.edu.cn`
- 教务系统（登录入口）：在教务处官网或信息门户 `i.xtu.edu.cn` 进入。
- 校外访问：先连 **VPN**，再登录教务系统。

## 三类课程说明

### 一、必修课

- 由培养方案规定，直接排入课表。
- 未修完必修课会成为“重修单”，直接影响毕业和学位。
- 通常无需在选课窗口主动选择，但要留意 “**教室、时间、任课老师**”，避免时间冲突。

### 二、专业选修课

- 培养方案预设选修课清单，按学期开放。
- 每学期有明确的选课学分下限，达不到会影响毕业。
- 部分课程有先修要求，例如“完成 XX 后才能选 YY”。
- 选课时优先选 **本专业核心方向** 相关课程，兼顾兴趣发展。

### 三、通识课 / 自主发展课程

- 面向全校学生开放，涵盖人文、社科、艺术、理工、创新创业、思政等模块。
- 教务处每学期发布《关于做好 XXXX-XXXX-X 学期自主发展课程选课通知》和 **《湘潭大学教务系统选课操作手册》**。
- 常见附件：
  - 《关于做好 X 学期自主发展课程选课通知.docx》
  - 《湘潭大学教务系统选课操作手册.doc》
- 建议提前 30 分钟登录系统，做好选课策略（优先目标 → 备选 → 冷门保底）。

## 选课基本步骤（通用）

具体操作以当学期附件为准，通用步骤：

1. **登录教务系统**：用学号 + 密码登录。
2. **查看选课通知**：确认选课开放时段、可选模块、学分上限。
3. **选择课程**：进入“选课” → 按分类查看课程 → 加入待选。
4. **提交选课**：点击“提交”后，系统会返回选课结果。
5. **确认结果**：在“个人课表”查看是否选中；如未选中，进入补选或退选窗口。
6. **补 / 退选窗口**：一般开学前几天或第一周开放，具体以通知为准。

## 常见踩坑与提醒

- ❌ “等系统不忙再选”：热门课程秒空，建议提前 5–10 分钟就位。
- ❌ “选了不参加也没关系”：选而不学、连续缺勤会影响成绩，甚至进违纪流程。
- ❌ “退课随时都能退”：每学期有明确的退课截止时间，过时不受理。
- ❌ “校外能直接选”：多数情况下选课需要连接校园网或走 VPN，不要在没网的情况下卡在提交页。

## 面向新生的策略

1. **大一第一学期**：跟着培养方案走，通识课 / 自主发展课程选 1–2 门就够，别贪多。
2. **大一第二学期开始**：可以按兴趣和长期规划选一些跨学院通识，比如经济、法律、艺术类。
3. **大三之前**：完成培养方案里对通识 / 自主发展课程学分下限的要求，避免大四被动。
4. **想保研、竞赛、双学位的**：把有加分认可的课程（如学科基础课、跨学院拓展课）优先安排。

## 查通知与手册的入口

- **教务处官网通知栏**：`https://jwc.xtu.edu.cn`，选课季会集中发布通知。
- **教务系统首页公告**：登录后能看到最新通知。
- **辅导员群 / 学院教务办**：会转发通知；操作出问题时最快能解决的通常是学院教务老师。
