# 教学楼分布

湘大校园较大，教学楼、学院楼、公共教学建筑较多，第一次进校经常会遇到 “**知道楼名，不知道位置**” 的问题。这份指南先按 **片区 + 建筑清单** 梳理常见名称，方便查找。

## 一句话理解

- 主要教学与科研楼群聚集在 **兴湘片区** 和 **中心区**。
- 常用公共教学楼是 **第一教学楼、第三教学楼、南山阶梯教室、北山阶梯教室** 等。
- 各学院拥有自己的独立学院楼或办公楼，通常与其宿舍片区距离较近。

主要建筑包括：

- 环境与资源学院（环资院）楼
- 法学学部
- 研究生院
- 信息科技大楼
- 自动化与电子信息学院（自电院）楼
- 数学院（数学与计算科学学院）楼
- 材料院（材料科学与工程学院）楼
- 分析测试中心
- 物理院（物理与光电工程学院）楼
- 工科楼
- 信息实验楼
- **南山阶梯教室**

## 常用步行提醒

- 兴湘片区 → 中心区通常步行 15–20 分钟，或坐校内代步工具节省时间。
- 期末周图书馆和阶梯教室占座紧张，需要提前踩点熟悉入口和大教室位置。
- 大教室常同时承担多门课，进错门在湘大是常见小尴尬，建议养成 “看门牌 + 对课表” 的习惯。

## 使用小建议

1. 迎新期建议携带的两个 “导航神器”：
   - 高德 / 百度地图搜 “湘潭大学” 定位后叠加校内步行路径。
   - 官方 720 全景图（见 [湘大校园地图](/library/campus/湘大校园地图)）。
2. 第一次上课前一天可以按课表把 “教室 → 建筑 → 片区” 走一遍，避免第二天迟到。
3. 高年级课程会集中在本学院楼，尽量选靠近本学院楼的宿舍片区，能节省大量通勤时间。
4. 涉及 “某栋楼几层有饮水机、几层是教师办公”、“空教室能不能借用” 等问题，以本院教务办和后勤保障部通知为准。
