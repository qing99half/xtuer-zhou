# 学校简介及校史

学校简介-湘潭大学

湘潭大学（Xiangtan University），简称“湘大”，位于湖南省湘潭市，是毛泽东同志亲自倡办的综合性全国重点大学、国家“双一流”建设高校。

学校创办于1958年，同年9月10日毛泽东同志亲笔题写“湘潭大学”校名。1959年因故停办；1974年获批复校，定位为文、理、工综合性大学；1978年被国务院确定为综合性全国重点大学；1981年成为全国首批硕士学位授权单位；2022年入选国家“双一流”建设高校。
