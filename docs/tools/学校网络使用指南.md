# 学校网络使用指南

湘大校园网 **实名制**，全体学生自动分配 **学号** 作为上网账号；掌握账号规则和绑定流程后，才能顺畅使用校内网络和外网。

## 一句话理解

- 学生上网账号 = **学号**；教职工 = **工号**。
- 初始密码常见格式：**Xtu@入校年份 + 身份证后六位**（旧版直接身份证后六位）。
- 学生账号最多 **2 台设备同时在线**，超限会强制下线。
- 学生通用 WiFi 名称：**XTU-STUDENT**。

## 一、账号基础规则

### 1. 实名制账号

- 全体学生自动分配学号作为上网账号；教职工用工号。
- 教学区、图书馆、食堂等公共区域 **XTU-STUDENT 免费无线网**，仅需学号认证即可使用内网、知网、选课系统。

### 2. 初始密码

- **旧版**：身份证后六位。
- **新版**：`Xtu@入校年份 + 身份证后六位`。
- 首次登录 **强制改密码**；要求 8 位以上、大小写 + 数字 + 特殊符号；**每半年必须更换**。

### 3. 同时在线设备限制

- 学生账号最多 **2 台** 终端同时在线（手机 / 电脑 / 平板任意组合）。
- 教职工最多 **4 台**。
- 超限直接强制下线；频繁超限会账号拉黑断网。

### 4. 账号权责

- 账号仅限本人使用，严禁 **出借、出租、共享** 给室友或校外人员。
- 账号产生的所有网络违规、法律责任由 **账号实名本人** 承担。

## 二、校园网绑定流程

1. **登录自助中心**：账号一般为学号；密码为 `Xtu@入学年份+身份证后六位` 或身份证后六位；忘记密码可拨打 **58293315** 重置。
2. **进入 “我的套餐”**，变更校园网套餐。
3. **点击 “运营商管理”** → 添加号码：账号是办理好的电话号码，运营商密码是 `123456`。
4. 若遇到其他问题可联系对应运营商。

## 三、登录 / 掉线常见规则

1. **非正常下线**（直接关 WiFi、休眠断网）会造成账号 “挂在线上”，再次登录提示超限。等待 **15 分钟** 自动释放即可重连。
2. **无线名称**：学生通用 **XTU-STUDENT**，无其他校内免费 WiFi。
3. **离校假期**：运营商宽带可暂停，校内免费网寒暑假正常开放。

## 四、常见场景怎么办

### 场景 A：宿舍宽带上不了网

1. 先自行 **重启光猫 + 路由器**。
2. 无效则联系电信校内专属客服，**24 小时内** 会安排工作人员上门检修。
3. 校园网后台故障，前往 **南苑网络与信息中心** 登记报修。

### 场景 B：切换设备后无法登录

- 可能是同时在线设备超限。
- 等待 15 分钟自动释放或手动踢下其他设备。

### 场景 C：忘记密码

- 拨打 **0731-58293315** 重置。
- 或到 **南苑网信中心服务窗口** 现场处理。

### 场景 D：宿舍宽带迁移

- 同园区换宿舍：凭宿舍调整证明到 **营业厅** 办理宽带地址变更。
- 跨园区调换：需提交申请，**1–3 个工作日** 完成迁移。

### 场景 E：账号封禁

- 携带身份证、办理回执到电信校内营业厅申请解封。
- 后续严格遵守 “一人一号” 使用规则。

## 五、咨询联系方式

- 网信中心综合咨询：**0731-58293315、58298997**
- 校园网缴费 / 账号问题：**0731-58292559**
- 联通校内宽带专线：**0731-52872000**
- 线下办理：**南苑校园卡服务点、三大运营商校内营业厅**

## 六、常见误区

- ❌ “账号可以借室友用一下”：违规，出问题账号持有人担责。
- ❌ “只用免费 WiFi 就够了”：只能应急，无法支撑网课、网游、大型软件下载。
- ❌ “密码不用改”：首次登录必须改密码，且每半年更换一次。
- ❌ “直接关电脑就退出登录了”：非正常下线会导致 15 分钟内无法重登。

## 使用小建议

1. 报到第一天就把 **校园网密码修改** 完成，避免 “忘记密码 + 输错锁定” 双重卡壳。
2. 手机端记得开启 **自动重连 XTU-STUDENT**，日常上网无缝切换。
3. 涉及电信 / 联通 / 移动的宿舍宽带办理，见 [校园电话卡与寝室宽带办理](/library/tools/校园电话卡与寝室宽带办理)。
4. 校外访问校内资源，请使用 [VPN 使用指南](/library/tools/VPN使用指南)。
