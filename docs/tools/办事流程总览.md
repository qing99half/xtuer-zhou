# 校园常见办事流程

大学四年经常会遇到 “**这个事该找谁**” 的问题。这份指南先把新生常见的办事事项列出来，指清 “**流程归属 + 常见节点**”，方便对号入座。

## 一句话理解

- 教务类找 **教务处 / 学院教务办**。
- 学工类找 **学工部 / 学院学工办 / 辅导员**。
- 生活类找 **后勤保障部 / 各片区物业**。
- 校园卡 / 一卡通类找 **一卡通中心 / 财务处**。
- 网络 / 门户 / 系统类找 **信息化建设与管理处 / 信息中心**。

## 常见办事清单

### 一、学生证补办

- 归属：**教务处**。
- 常见流程：
  1. 前往学院教务办登记信息。
  2. 携带身份证、近期免冠照片到教务处指定窗口办理。
  3. 缴纳工本费（金额以当年通知为准）。
  4. 领取新学生证；如需盖章、贴车票优惠章，按窗口指引办理。

### 二、考试违纪处分流程

- 归属：**教务处 / 学生工作部**。
- 常见流程：
  1. 监考老师发现违纪 → 现场记录并上报。
  2. 学院 / 教务处组织谈话 → 学生陈述。
  3. 学校学位委员会 / 学生工作部依规审议 → 出具处分决定。
  4. 处分决定送达 → 学生可在规定期限内提出申诉。
- 违纪代价：轻则警告 / 记过，重则留校察看、开除学籍，并可能影响学位授予。

### 三、保研 / 推免流程

- 归属：**教务处 + 各学院**。
- 详情见 [保研与科学竞赛](/library/study/保研与科学竞赛)。

### 四、课程认定 / 学分认定流程

- 归属：**教务处 + 学院学位委员会**。
- 常见流程：
  1. 修读课程完成后取得成绩单、课程大纲。
  2. 填写《境外 / 校外修读课程学分认定申请表》。
  3. 学院学位委员会审核。
  4. 教务处备案，学分录入教务系统。

### 五、选课流程

- 归属：**教务处 + 学院教务办**。
- 详情见 [选课与通识课](/library/study/选课与通识课) 与 [选课、修课与访学](/library/study/选课修课与访学)。

### 六、请假 / 销假

- 归属：**辅导员 / 学院学工办 / 教务办**。
- 常见流程：
  1. 提前填写请假申请表，说明事由和请假时间。
  2. 由辅导员 / 学院负责人审批。
  3. 请假期间保持联系；返校后销假。

### 七、宿舍问题（换宿舍、维修、报修）

- 归属：**后勤保障部 / 宿舍片区物业**。
- 常见流程：
  1. 报修：登记 / 报修 App / 微信小程序 / 前台。
  2. 更换宿舍：走后勤保障部审批流程，需提供事由说明。

### 八、一卡通问题（充值、挂失、补办）

- 归属：**一卡通中心**。
- 常见流程：详见 [校园一卡通指南](/library/tools/校园一卡通指南)（后续会补齐）。

### 九、缴费问题（学费、宿舍费、网络费）

- 归属：**财务处 / 学校统一缴费平台**。
- 常见流程：详见 [线上缴费指南](/library/tools/线上缴费指南)（后续会补齐）。

### 十、故障申报（网络、水电、系统）

- 归属：**信息化建设与管理处 / 后勤保障部**。
- 常见流程：详见 [故障申报与受理](/library/tools/故障申报与受理)（后续会补齐）。

### 十一、转专业

- 归属：**教务处 + 原学院 + 目标学院**。
- 详情见 [转专业指南](/library/study/转专业指南)。

### 十二、奖助学金申请

- 归属：**学生工作部 / 各学院学工办**。
- 详情见 [奖学金与助学金政策](/library/tools/奖学金与助学金政策)。

### 十三、休学 / 复学 / 退学

- 归属：**教务处 + 学工部 + 学院**。
- 常见流程：需填写申请表、提交证明材料（如医疗证明、家庭事由证明等），由学院初审、学校审批，办理离校 / 复学手续。

### 十四、党团组织关系转接

- 归属：**校党委组织部 / 校团委 / 各学院党团组织**。
- 常见流程：入学时提交党团材料 → 组织关系转入 → 毕业时按规定转出。

## 通用建议

1. **办事前先确认归属**：找错部门是最常见的时间浪费。
2. **准备好材料清单**：多数流程涉及身份证、学生证、成绩单、申请表等，出门前一次带齐。
3. **注意办事时间**：多数窗口有工作日办公时间，节假日不办公。
4. **留存凭证**：申请表、审批表、缴费凭证都建议保留电子版和纸质版。
5. **紧急事项走辅导员**：辅导员通常最了解校内各类流程，是新生最好的“路由器”。
