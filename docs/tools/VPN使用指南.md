# VPN 使用指南

湘大校园 VPN 覆盖 **全体在校本科生、研究生、教职工**，是校外访问知网、教务、学工系统等 **校内资源** 的必备工具。这份指南把两条 VPN 通道 + 关键规则 + 常见故障讲清楚。

## 一句话理解

- **入口**：`https://vpn.xtu.edu.cn`。
- **账号**：学号 / 工号，与信息门户通用。
- 客户端 VPN 用 **EasyConnect**（长期使用）；网页 VPN 免安装（临时应急）。
- 严禁账号出借、爬虫下载、翻墙访问境外非法网站，违规会 **封号**。

## 一、基础信息

### 使用人群与权限

- 全体在校 **本科生、研究生、教职工** 免费使用。
- 账号密码 = 信息门户账号密码（学号 / 工号登录）。
- **毕业离校后账号自动注销**，无法继续使用。

### 校外主要用途

在校外家里、外地、高铁等外网环境，访问仅限校内的系统：

- 图书馆 **知网、万方、外文文献数据库**
- **教务系统、选课、成绩查询、学工系统**
- 校内 **办公平台、政务信息港、校内论文系统**

### 两类通道（二选一，不能同时开）

- **SSL VPN**（客户端 EasyConnect）：推荐电脑长期使用。
- **WEB VPN**（网页免安装）：临时手机快速用。

## 二、SSL VPN（EasyConnect 客户端）

### 服务器地址（3 个备用）

- **首选域名**：`https://vpn.xtu.edu.cn`
- **备用 IP 1**：`https://202.197.228.8`
- **备用 IP 2**：`https://218.75.242.251`

> IP 地址打开会提示 “连接不安全”，忽略警告继续访问即可。

### 电脑端（Windows / Mac）

1. 浏览器打开 `https://vpn.xtu.edu.cn`，页面会自动匹配系统下载 **EasyConnect 安装包**。
2. 安装完成后打开软件，填入 **首选服务器地址**。
3. 账号 = 学号，密码 = 信息门户登录密码，登录连接。
4. 提示连接成功后，即可直接打开浏览器访问校内资源。

### 手机端（安卓 / iOS）

1. 手机浏览器打开 VPN 官网，下载对应移动端安装包（iOS 可直接商店搜 EasyConnect）。
2. 打开 App，填入服务器地址，登录学号密码。
3. 连接成功后，**手机浏览器切换桌面版** 才能正常下载知网文献。

## 三、WEB VPN（免安装网页版）

- **访问地址**：`https://wrdwebvpn.xtu.edu.cn`
- **登录方式**：信息门户账号密码网页登录，无需装软件。
- **导航栏**：内置图书馆数据库、信息门户一键跳转，也可手动输入校内网址。
- **限制**：
  - 校园内网环境无法打开。
  - 和客户端 VPN 不能同时在线。

## 四、官方硬性使用规定（违规封号）

1. **账号仅限本人**：严禁转借、出租、分享给校外人员；所有违规责任由账号持有人承担。
2. **文献下载禁令**：禁止批量、高速爬虫下载知网 / 外文数据库；过量下载会冻结 VPN 账号，严重需赔偿数据库年费损失。
3. **禁止搭建代理、热点分流 VPN 网络**：网信中心检测到直接封禁外网权限。
4. **仅限校内学习办公**：严禁翻墙访问境外非法网站，一经发现通报学院 + 校规处分。
5. **单人在线**：同一账号仅单人在线，多设备同时登录会强制下线。

## 五、高频故障解决

### 提示无法连接服务器

- 检查地址必须带 `https://`。
- 切换 **备用 IP** 重试。

### 登录成功但打不开校内网站

- 清理浏览器缓存。
- 手机浏览器切换 **桌面模式**。

### 频繁自动掉线

- 卸载旧客户端，官网重新下载最新版。
- 切换稳定 WiFi / 流量。

### 知网无法下载文献

- 手机务必开启浏览器 **桌面版**。
- 电脑端直接用客户端 VPN 访问。

## 使用小建议

1. 到校第一周把 **EasyConnect 客户端** 装好、测试一次，避免寒暑假回家临时抓瞎。
2. 用完 VPN **主动断开连接**，不然会占用同时在线名额。
3. 文献下载采用 **人工节奏**，不要一次性大批量下载；也不要写脚本爬。
4. 校外访问校内系统只走官方 VPN；不要用不明来源的 “替代方案”。
5. 涉及具体版本 / 界面截图，请以网信中心官网、官方公众号最新版本为准。
