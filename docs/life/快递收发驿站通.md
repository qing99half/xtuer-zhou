# 快递收发驿站通

湘大新生入学后，**寄快递、收快递** 是每周高频动作。这份指南把 **收件地址、驿站分布、常见快递品牌** 一次讲清楚。

## 一句话理解

- 学校邮寄地址：**湖南省湘潭市雨湖区北二环羊牯塘街道湘潭大学**。
- 校园主要有 **东门菜鸟驿站、东门速递易、琴湖菜鸟驿站、琴湖乐收服务站** 等驿站。
- 不同快递品牌对应不同驿站，寄件人务必写对目标驿站名称。

## 学校寄件地址

- **地址**：湖南省湘潭市雨湖区北二环羊牯塘街道湘潭大学。
- **收件人**：填写你的姓名 + 学院 / 宿舍片区。
- **电话**：填写你本人手机号（用于送达时通知）。

⚠️ 有些快递会精确到宿舍楼栋，建议提前弄清楚自己片区名称（西湖、琴湖、金翰林、南苑、北苑、兴湘）。

## 校园快递驿站分布

### 东门附近

- **东门菜鸟驿站**：**中通快递**。
- **东门速递易服务站**：**邮政 EMS 快递**。

### 琴湖片区

- **琴湖菜鸟驿站**：**极兔、韵达、圆通、申通、京东** 快递。
- **琴湖乐收服务站**：**顺丰快递**。

### 其他片区

- 金翰林、南苑、北苑、兴湘等片区一般也有对应的驿站或代收点，具体以现场为准。
- 如果不确定自己的快递到了哪个驿站，可以在 App（如菜鸟裹裹、京东、顺丰）查询实际派送信息。

## 寄件常见问题

### 1. 寄件比取件麻烦吗？

- 常规寄件走驿站或快递员上门。
- 顺丰、京东等品牌通常可预约上门取件，也可以在小程序 / App 下单。

### 2. 双十一 / 618 期间会不会爆仓？

- 大促期间驿站积压严重，注意 **及时取件**，避免超时收费。
- 部分驿站会临时增加派件时间，请留意公告。

### 3. 遗失、破损怎么办？

- 拆件前先 **拍视频记录外包装**，如发现破损立即联系快递员或驿站。
- 网购的商品还可以联系电商平台申请赔付。

### 4. 校内其他代收点？

- 部分片区有代收小卖部、菜鸟自提柜等，具体以现场为准。
- 疫情、开学等特殊时期可能会临时调整派送路径，请以现场通知为准。

## 使用小建议

1. 报到前把大件生活用品寄到学校快递驿站，比拎去学校更省力。
2. 收件人一栏尽量精确到 “**学院 + 宿舍片区 + 手机号**”，避免同名同姓混淆。
3. 大件快递（被褥、书籍）建议约定 **顺丰 / 京东** 等能追踪的品牌。
4. 快递到达通知一般会通过 App / 短信推送，注意勾选微信订阅。
5. 涉及具体驿站地址、开放时间、代收费用，请以现场公告为准。
