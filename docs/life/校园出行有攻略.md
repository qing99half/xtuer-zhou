# 校园出行有攻略

湘大校园大，进出校园的方式多样。这份指南把 **校内交通 + 校外交通 + 湘潭市内公交** 三部分讲清楚，方便你按需选择。

## 一句话理解

- 校内主要坐 **“小白”**（校园代步车），单程 1 元。
- 长沙 → 湘潭：**长株潭城铁 20 元 / 高铁 12 元 / 长沙地铁 4 号线 + 3 号线 + 西环线全程 8 元**。
- 出高铁站可以坐 **K1 专线** 或迎新大巴直达湘大东门。

## 一、校内交通

### 校园小白

- **单程价格**：1 元。
- **停靠点**：西湖、南门、琴湖、东门、北苑、图书馆、一教、三教、逸夫楼、工科楼、南山阶梯、南苑等多个地点。
- **叫车方式**：使用 “**青程校园派 App**” 在 “小白” 停靠点呼叫车辆。
- **温馨提示**：
  1. **新生迎新期间可以免费乘坐 “小白”**。
  2. 小白座位前方会贴有二维码，微信扫码即可支付乘车费用。
  3. 不同的小白车辆有不同的路线规划，上车前问一问师傅是否途经目的地。

## 二、校外交通（长沙 → 湘潭）

### 方案 A：长沙火车站 → 长株潭城铁

- 从长沙火车站坐 **长株潭城铁**。
- 车次以 **“S” 开头**，可在 “**铁路 12306**” App 或微信小程序购票。
- 全程约 **1 小时**、单程 **20 元** 左右，直达 **湘潭站**。

### 方案 B：长沙南高铁站 → 高铁

- 从长沙南高铁站坐高铁。
- 全程约 **10 分钟**、票价 **12 元** 左右，到达 **湘潭北站**。

### 方案 C：黄花机场 → 磁悬浮 → 高铁

- 乘机场磁悬浮到长沙南站。
- 再坐高铁到湘潭北站。
- 最后坐公交车到湘大。

### 方案 D：机场大巴（可能需要提前联系）

- 部分时段有机场大巴直达学校。
- 具体班次和上下车点请提前咨询。

### 方案 E：地铁 + 城际（全程 8 元）

- **长沙南站**（高铁 / 火车抵达后）→ 换 **长沙地铁 4 号线**（往罐子岭方向）→ 至 **阜埠河** 站换 **3 号线**（往山塘方向）→ 至 **山塘** 站换 **长株潭西环线**（往湘潭北站方向）→ **湘潭北站**。
- 全程约 **1 小时**、**8 元**。
- 详见 [湘潭地铁乘坐指南](/library/life/湘潭地铁乘坐指南)。

## 三、湘潭站 → 湘大

- **公交**：从湘潭站出站往公交停靠站走，可以乘 **17 路公交车** 或迎新车辆，直达学校，在 **湘大体育馆东公交站** 下车，用时约 **40 分钟**。
- **打车**：直达寝室楼，价格约 **20–30 元**。
  - 开学季比较难打到，且学校可能会限制外来车辆进入。
- **出租师傅揽客**：站口会有很多师傅揽客，开学季可能涨价（一般按人头算），可以适当砍价。

## 四、湘潭北站 → 湘大

- **专线 / 迎新车辆**：出高铁站可以直接乘坐 **K1 专线** 或迎新车辆，在 **湘大东门公交站** 下车，约 **1 个半小时**。
- **打车 / 网约车**：门口出租会稍贵，价格约 **35–50 元**。

## 五、新生须知

1. 迎新期间，湘潭站和湘潭北站都会有志愿者举 “**湘潭大学**” 牌子，可以统一由大巴送到学校。
2. 湘潭公交需要下载 “**湘潭出行**” App，或使用支付宝出行领取电子公交卡。
3. 长沙公共交通：
   - 安卓：下载 “**潇湘一卡通**” App。
   - 苹果：使用钱包功能添加卡片，打开 NFC 即可使用（有一定折扣）。
4. 具体路线可以借助 **高德地图** 或 **百度地图**。

## 六、按目的地选方案

| 目的地 | 推荐方案 |
| --- | --- |
| 长沙火车站 → 湘潭 | 长株潭城铁 S 车（20 元 / 1 小时） |
| 长沙南高铁 → 湘潭 | 高铁到湘潭北（12 元 / 10 分钟） |
| 长沙市区玩耍 | 城铁 + 地铁 或 西环线 + 地铁 |
| 机场直达湘大 | 磁悬浮 → 高铁 → 湘潭北 → 打车 |
| 湘潭站 → 湘大 | 17 路公交 / 打车 |
| 湘潭北站 → 湘大 | K1 专线 / 打车 |

## 七、常见问题

### Q1. 开学季打车很贵吗？

- 是。开学季外出车辆增加、供不应求，出租师傅可能按人头算价格，需砍价。
- 建议提前预约网约车或直接坐公交、迎新大巴。

### Q2. 从校园去湘潭站远吗？

- 从学校到湘潭站坐公交约 40 分钟，看你上车片区。

### Q3. 想去张家界 / 岳阳等地怎么走？

- 通常先到长沙南高铁站 / 火车站，再换乘高铁 / 火车。
- 具体线路以 12306 和火车站服务信息为准。

### Q4. 校园里可以骑共享单车吗？

- 校内通常有一定范围可以使用共享单车，具体以校园管理规定为准。

## 使用小建议

1. 出行前先确认目的地在长沙市内的位置，再选合适的换乘方案。
2. 长沙地铁 App、铁路 12306、高德 / 百度地图是新生必装 App。
3. 涉及具体票价、班次、停靠点，请以铁路 12306、地铁 App、公交现场为准。
4. 长途返家 / 返乡建议提前 **30 分钟** 到站，避开高峰期。
