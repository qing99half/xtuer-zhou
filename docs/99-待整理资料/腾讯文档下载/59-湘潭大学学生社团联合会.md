湘潭大学学生社团联合会
一、组织基本定位
湘潭大学学生社团联合会（现官方名称学生社团管理中心，简称“社管/社联”），是在校团委直接领导下，统筹全校学生社团的校级学生组织，是连接学校与百余学生社团的核心管理服务平台。
• 工作方针：服务、协调、监督、管理
• 工作理念：在沟通中了解，在了解中相融，在规范管理中发展
• 核心使命：规范社团运营、丰富校园文化，为全校学生打造多元课余成长平台
• 管理规模：统筹全校60余个注册学生社团，覆盖学术、文艺、体育、志愿、科创、传统文化六大类别
二、完整组织架构
顶层决策层
1.
督导团：由具备多年社团工作经验的高年级学生组成，负责监督、指导社联整体工作，把控发展方向
2.
常务委员会（主席团）：设主席1名、副主席2名、团总支书记1名，统筹全年工作规划、社团重大事项审批、对外对接
下设职能部门
1.
办公室（行政人事部）
行政中枢，负责会议统筹、档案物资管理、人员考勤、内部制度建设、社团文件收发存档。
2.
团务管理部
管理社团团支部，负责社团评优、社团负责人换届考核、新社团成立审批、社团年审、社团干部培训。
3.
财务部
统一监管全校社团经费，审核社团活动报销、活动预算，规范社团财务收支。
4.
策划部
策划统筹全校大型社团品牌活动（百团大战、社团风采月等），撰写活动方案、场地申请协调。
5.
新媒体运营部（网络信息部）
运营社联官方新媒体账号，活动摄影、推文排版、短视频制作，对外宣传社团文化。
6.
宣传部（广告组）
设计海报、展板、周边物料，线下活动视觉布置，校园线下宣传落地。
7.
公关部（外联部）
对接校外商家争取活动赞助，联动省内高校社联交流，统筹跨社团合作活动。
8.
多媒体技术组
提供活动灯光、音响、直播、视频剪辑等技术支持。
三、四大核心职能
1.
管理监督
制定社团管理规章，完成社团注册、年审、注销；审核社团活动、经费、场地；定期开展社团负责人座谈会，规范社团运营，杜绝违规活动。
2.
服务扶持
为社团提供场地、物资、宣传、经费支持；开展社团骨干培训、策划分享会；收集社团诉求，对接校团委解决社团发展难题。
3.
活动统筹
牵头举办全校性大型社团文化活动，整合各社团资源联合展演，搭建社团展示平台。
4.
文化引领
打造校园社团特色文化，推动学术、传统、科创、文体类社团均衡发展，丰富第二课堂。
四、标志性品牌活动
1.
百团大战（新生招新）
每年9月开学大型社团统一招新，60余个社团集中摆摊展示，配套“百团奇妙夜”晚会，是新生了解社团的核心活动。
2.
社团风采月/青年文化交流节
全年集中展示各社团特色：汉服走秀、乐器路演、书画创作、棋类对弈、体育展演、外语互动等沉浸式文化体验活动。
3.
迎新游园会
面向新生的趣味互动活动，联动社团开展小游戏、礼品打卡，快速帮助新生融入校园。
4.
社团评优表彰大会
年度评选优秀社团、优秀社团负责人、优秀会员，举办答辩表彰，树立社团标杆。
5.
社团骨干培训班
定期开展策划、财务、宣传、管理类培训，提升社团负责人综合能力。
五、校内社团分类（社联管辖）
1.
学术科创类：数学、法学、辩论、机器人、建模、编程等协会
2.
文艺艺术类：吉他、声乐、汉服、书画、戏剧、舞蹈、摄影等
3.
体育竞技类：篮球、滑板、羽毛球、武术、骑行、田径等
4.
传统文化类：国学、书法、围棋、非遗手工、传统礼仪协会
5.
公益志愿类：爱心助学、环保、敬老、支教志愿服务社团
6.
综合兴趣类：外语、桌游、推理、电影、心理、创业实践社团
六、与校学生会核心区别
组织 服务对象 核心工作
校学生会 全体在校学生 班级事务、学风建设、权益维权、全校文体赛事、学生生活服务
社团联合会 全校学生社团+社团会员 管理、扶持60余个社团，统筹社团文化活动，运营社团招新与展演
七、新生加入相关说明
1.
招新时间：每年9月百团大战现场统一招新，线上同步开放报名渠道
2.
选拔流程：现场报名/线上填表→一轮面试→部门分配，正式成为干事
3.
成长收获：锻炼活动策划、财务、新媒体、沟通统筹能力；学生工作经历可用于综测加分、评优评先；深度接触全校各类兴趣社团资源。
