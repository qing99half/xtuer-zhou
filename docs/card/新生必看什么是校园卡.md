# 新生必看！什么是校园卡

校园卡是新生入学后最先接触的实物证件之一，与校园身份、消费、借阅、缴费都相关。这份专栏目前仅为占位，主要指路到已入库的相关资料，等到源文档正文抓取到位后再逐段补齐。
