# 实验班与拔尖班选拔

湘潭大学有多个 **拔尖创新人才培养实验班**，是学校集中优质资源培养顶尖学生的通道。加入实验班对培养环境、师资、保研通道都有明显加持。这份指南先把清单列出来，再讲选拔和培养的通用做法。

## 一句话理解

- 实验班的核心特点：**小班化教学 + 双导师制 + 个性化培养 + 单独保研指标**。
- 选拔时机通常在 **高考录取** 或 **入学后大一/大二学期切换点**。
- 加入不难，留下来更看学习状态：多数班有年度考核和分流机制。

## 现有拔尖班一览

学校在原有 9 个拔尖创新人才培养实验班基础上继续扩容，公开信息中常见的实验班包括：

- **数学类韶峰班**
- **物理学韶峰班**
- **化学类韶峰班**
- **材料类师昌绪班**（院士任班主任）
- **化学工程与工艺韶峰班**
- **经济学类韶峰班**
- **公共管理类韶峰班**
- **马克思主义理论韶峰班**
- **法学卓越班**
- 新增：**汉语言文学韶峰班**、**计算机科学与技术韶峰班**
- 新增：**韶峰科创班**（面向选考物理科目学生，机器人应用—低空经济方向，采用“一校对多企”联合共建，校企双导师制）

具体开设情况每年会有调整，以本科招生网通知为准。

## 选拔方式

各拔尖班的具体选拔机制略有差异，但通用做法是：

- **面向新生 / 在校生**：部分实验班在高考录取阶段就单独招生；部分实验班在入学后组织二次选拔。
- **考察方式**：多为“笔试 + 面试”，考察综合素质、学术潜力、创新意识和相关基础学科能力。
- **选拔窗口**：新生入学后一段时间内组织选拔，具体时间以本科招生网、教务处、学院当年通知为准。
- **动态调整**：多数实验班有年度考核，不达标可能会被分流回原专业；达标的普通专业学生也可能通过考核进入实验班。

## 培养特色

- **一流师资团队**：由资深教授、院士级学者、青年学术骨干担任导师。
- **一流学习条件**：资源、图书、实验、科研经费上有专项支持。
- **一流学术环境**：接触国家级课题、跨学科项目、访学交流机会。
- **导师制**：普遍推行 **学业导师 + 学术导师双导师制**。
- **升学效果**：以数学类韶峰班、材料类师昌绪班等为例，公开报道显示这些班的保研升学率明显高于全校平均。

## 常见问题

### 1. 高考录取时没进拔尖班，入学后还能进吗？

多数拔尖班有入学后二次选拔通道，通常在大一或大一到大二切换阶段组织。具体是否面向普通生开放要看每个实验班当年的通知。

### 2. 实验班和普通班的课不一样吗？

实验班通常执行 **独立培养方案**，有单独的必修课、专业主干课，甚至独立的教学团队和课程节奏。部分公共基础课可能与普通班合班，但专业核心课通常小班独立开设。

### 3. 进了实验班一定能保研吗？

不是自动保研。实验班通常有 **单独的推免专项名额**（如经济学类韶峰班每年 10 个），但需要满足推免的资格条件（绩点、加分、无违纪等），也要通过学院和学校的推免评审。

### 4. 分流后会被贴“淘汰”标签吗？

分流本身是正常的动态调整机制，学生回到原专业后继续按原培养方案学习，不会影响毕业和常规评奖资格。

### 5. 微专业和拔尖班是一回事吗？

不是。**微专业** 是学校在优势学科方向推出的辅修/短周期专业方向，面向全校学生自愿修读，不同于拔尖班的建制化人才培养。

## 使用小建议

1. 报到时留意学校迎新材料里的 **实验班宣讲会 / 招募通知**，通常会在入学第一个月内密集发布。
2. 想冲拔尖班的同学可以提前联系学长学姐或目标学院辅导员，了解笔试内容偏基础还是偏拓展。
3. 进了拔尖班后要跟紧课程节奏，不适应时及时与学业导师沟通，避免年度考核不达标被动分流。
4. 无论是否进拔尖班，都要认真准备大一课程；很多同学是通过大一 GPA + 学院推荐进入实验班的。
5. 涉及具体的“今年招几个人 / 什么时候考试 / 面试问什么”，一律以当年官方通知为准。
