# 培养计划

“培养计划”，又叫“培养方案”，是学校对每个本科专业整个 4 年（或 5 年）里 **要学什么、要修多少学分、什么时候修** 的完整规定。它是选课、绩点计算、毕业审核、保研评估的“底稿”，新生入学后要拿到并阅读一遍。

## 一句话理解

- 培养方案 = 你 4 年内的“课程地图”。
- 每门课、每个学分、每次实习实训都写在里面。
- 选课、修辅修、参加拔尖班、转专业，都会用这份文件反查。

## 培养方案主要写了什么

一份典型的湘大本科专业培养方案通常包含以下模块：

1. **专业培养目标**：这个专业要培养什么样的人、就业方向是什么。
2. **毕业总学分要求**：需要修满多少学分才能毕业、拿学位。
3. **课程结构**：把课程分成几大类，通常包括——
   - 公共基础课（A 类 / B 类，比如思政、体育、外语、数学、物理等）
   - 专业基础课
   - 专业主干课
   - 专业选修课
   - 毕业设计 / 毕业论文 / 实习实践
4. **学期分布**：每学期建议修哪几门课、什么时候开专业实验、什么时候开始毕设。
5. **附加说明**：辅修可选清单、拔尖班培养要求、双学位安排等。

学分构成示例（以经济学专业本科培养为例）：公共课 42 学��、专业基础课 30 学分、专业主干课 14 学分、专业选修课不少于 14 学分、毕业设计或其他毕业实践 17 学分，总学分不少于 117 学分；不同专业要求会有所不同，你所在专业以本院培养方案为准。

## 在哪里能拿到本专业培养方案

- **湘潭大学教务处 → 培养方案**：`https://www.xtu.edu.cn/jyjx/bksjyjx/pyfa.htm`
- **湘潭大学官网 → 本科生教育教学 → 培养方案**（含公共基础 A 类、B 类课程附件）：`https://www2024.xtu.edu.cn/jyjx/bksjy/pyfa.htm`
- **你所在学院官网**：多数学院会在“本科生教育 / 教学 / 培养方案”栏目单独发布，如自动化与电子信息学院、商学院、物理与光电工程学院等，路径类似 `<学院拼音>.xtu.edu.cn`。
- **迎新期材料**：报到时通常会发放本专业培养方案纸质版或电子版。

小提醒：培养方案会区分“年级版本”，例如“2025 级培养方案”只适用于该级新生，学长学姐的版本不能完全套用。

## 培养方案的实际用法

### 1. 选课时

选课系统里，能选的课基本对应培养方案里的课程清单：必修课系统会自动排入课表，公共课、专业选修课需要自己在选课时段选。有些课在培养方案里注明“限本专业” 或“先修某门课”，超前选或错过学期都会遇到麻烦。

### 2. 算学分 / 绩点时

- 每门课的学分、必修/选修属性，都对应培养方案里的分类。
- 未修完的必修课会成为“重修单”，直接影响毕业和学位。
- 保研、评优、评奖学金时看的是 **总加权绩点**，也是按培养方案里的学分权重来算的。

### 3. 修辅修 / 双学位时

湘大自 2009 年起推行主辅修制。辅修学位教育实行学分制管理，学制 2 年（4 学期），课程门数不低于 10 门、不低于 50 学分。跨学科门类修读者，符合主修学位授予条件后可获辅修学位证书。学有余力的学生可以在大一、大二结束前申请辅修。

### 4. 转专业时

湘大转专业的基本原则是 “**转出无限制，转入要考核**”。艺术类专业不能转入普通类专业（艺术学院内部除外），中外合作办学学生不能转专业。转专业成功后，你会切换到新专业的培养方案，之前修过的课按新方案换算是否可以抵学分。

### 5. 拔尖 / 实验班

湘大目前设有多个拔尖创新人才培养实验班（如数学类韶峰班、物理学韶峰班、化学类韶峰班、材料类师昌绪班、法学卓越班、经济学类韶峰班、公共管理类韶峰班、化学工程与工艺韶峰班、马克思主义理论韶峰班等），2025 年新增“韶峰科创班”面向选考物理科目学生选拔。实验班有独立培养方案，通常执行小班教学、双导师制和个性化培养。

## 新生该怎么读这份文件

拿到培养方案后，可以按下面顺序阅读，避免被密密麻麻的表格劝退：

1. **看总学分要求**：毕业至少要修多少学分，心里有个总账。
2. **看每学期开的必修课**：这就是接下来 4 年课程的“大纲版本”。
3. **标记专业选修课选修方向**：很多方案里会分“方向 A / 方向 B”，可以根据兴趣、就业方向做长期规划。
4. **记住实践、实习和毕设的时间点**：大三下、大四是集中期，别到时候临时抱佛脚。
5. **对照绩点公式**：培养方案里通常有绩点算法，保研、出国、评奖学金都用得到。

## 常见误区

- ❌ “拿到课表就够了”：课表只是当前学期视图，培养方案才是 4 年全景。
- ❌ “学长学姐的方案能直接用”：培养方案分年级版本，年级不同细节会变。
- ❌ “辅修就是多上几门课”：辅修有单独学分要求（不低于 50 学分），成本高、需要认真评估。
- ❌ “选修课随便选”：培养方案里有选修学分下限，达不到会影响毕业。

## 需要人问的时候

- 培养方案本身内容 → 找 **学院教务办 / 辅导员**
- 选课 / 排课 / 学分抵扣 → 找 **教务处**
- 拔尖班 / 微专业 / 韶峰科创班 → 关注 **本科招生网** 和所在学院通知
- 辅修专业申请 → 参照 [辅修教育 - 湘潭大学](https://www2024.xtu.edu.cn/jyjx/bksjy/fxjy.htm) 页面提示流程

无论看到哪一版培养方案，都记住一个原则：**以你所在学院、你所在年级的官方版本为准**。
