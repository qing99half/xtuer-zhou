# 湘大教务系统使用入门

教务系统是本科生日常最高频使用的系统之一，覆盖 **课表、成绩、选课、考试、培养方案** 等核心信息。这份指南先讲“怎么打开、怎么登录、怎么用”，避免新生第一次登录就懵。

## 一句话理解

- 教务系统入口通常放在 **教务处官网** 或 **信息门户** 后面。
- 常用账号是 **学号 + 初始密码**，第一次登录建议立刻改密码。
- 涉及“成绩、选课、考试”通常都要走教务系统，微信 / 抖音上流传的截图不代表官方数据。

## 系统入口

- **教务处官网**：`https://jwc.xtu.edu.cn`
- **信息门户**：`https://i.xtu.edu.cn`（登录后在应用中心找到教务系统）
- **校外访问**：需要连 **VPN**，再登录。
- **移动端**：部分功能通过学校官方 App（如“今日校园”“湘大 e 家”等，具体以学校当年推荐的应用为准）访问。

## 登录与账号安全

1. **学号**：录取通知书或迎新材料上会给出。
2. **初始密码**：常见做法是 “**身份证后几位**” 或按迎新说明设定，具体以官方通知为准。
3. **首次登录**：
   - 修改密码，采用 “**字母 + 数字 + 特殊字符**” 组合。
   - 绑定手机号（如系统支持），方便找回。
4. **忘记密码**：走系统的“找回密码”流程；如无法自助解决，联系学院教务办或教务处系统管理员。

## 常用功能一览

- **个人课表**：按学期、按周、按日查看，可导出打印。
- **成绩查询**：按学期查看，成绩单可申请打印盖章。
- **选课**：包括必修课自动排入、专业选修、通识 / 自主发展课程选课。
- **考试安排**：期末考试时间、教室、监考。
- **培养方案 / 教学计划**：查看本专业当前年级的培养方案。
- **网上评教**：期末通常要求学生对本学期任课教师进行网上评教。
- **申请类操作**：如学分认定、成绩复核、部分学籍变动申请。

## 通用操作步骤（示例）

1. **打开教务处官网**：`https://jwc.xtu.edu.cn`。
2. **找到教务系统入口**：通常在首页顶部导航栏或“常用链接”。
3. **登录**：输入学号 + 密码。
4. **进入个人主页**：可看到课表、成绩、通知等。
5. **完成目标操作**：选课 / 查成绩 / 打印课表 / 评教等。
6. **退出登录**：使用完毕主动退出，尤其在公共电脑上。

## 常见问题

### 1. 无法登录 / 显示密码错误怎么办？

- 确认学号大小写、数字位数。
- 尝试初始密码是否正确（身份证后几位或迎新说明中的默认密码）。
- 如果多次尝试失败被锁，联系学院教务办或教务处解锁。

### 2. 页面打不开怎么办？

- 检查是否在校园网内；不在校园网内要连 VPN。
- 使用推荐浏览器（Chrome / Edge 通常兼容较好）。
- 清除缓存后重新登录。

### 3. 选课页面卡顿怎么办？

- 高峰期正常现象。
- 建议提前 5–10 分钟登录，并准备好目标课程清单。
- 不要重复刷新页面提交订单，避免造成账户锁定。

### 4. 成绩看不到 / 有异常怎么办？

- 成绩上传由任课老师完成，学期末会陆续录入，不会全部同时发布。
- 如认为成绩明显有误，走 **成绩复核** 流程，具体窗口以教务处通知为准。

## 使用小建议

1. 每学期开学第一天就登录一次教务系统，确认自己的账号、课表、培养方案正常。
2. 手机上收藏教务处官网、信息门户、VPN 使用指南三个入口，方便随时进入。
3. 关键操作后（如选课提交、评教提交）保存截图，避免出现异常时无从举证。
4. 不要相信任何非官方渠道要求你 “填学号密码 / 帮你选课”，注意账号安全。
5. 遇到系统性问题（登录、页面、选课冲突），最先联系 **学院教务办** 或 **教务处**，不要在群里乱猜。
