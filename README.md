# xtuer-zhou 湘潭大学学习文档知识库

这是一个面向湘潭大学学习与校园资料沉淀的知识库项目。

## 当前定位

项目已进入 Next.js 网页站点开发阶段，第一版围绕首页、入学指南、资料库和后续 RAG 问答能力搭建基础骨架。资料仍坚持先进入待整理区，经过用户审核后再进入正式资料库或 RAG 索引。

## 目录结构

```text
docs/
├── 00-知识库说明/
├── 01-新生入学/
├── 02-课程学习/
├── 03-考试考证/
├── 04-学院专业/
├── 05-校园生活/
├── 06-办事指南/
├── 07-资源工具/
└── 99-待整理资料/

assets/
├── images/
├── files/
└── references/

product/
├── PRD.md
└── development-log.md
```

## 工作方式

- 每次 PRD 轮询确认后，更新 `product/PRD.md`。
- 每次发生实质变更后，更新 `product/development-log.md`。
- 原始资料先放入 `docs/99-待整理资料/`，整理后再归档到对应栏目。
