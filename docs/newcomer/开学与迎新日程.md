# 开学与迎新日程

湘大新生的开学季通常集中在 **9 月初**。这份指南按 “**报到 → 开学典礼 → 正式上课**” 的顺序把关键日程说明清楚，方便到校前后按节奏走。

## 一句话理解

- 报到日通常安排在 9 月上旬，具体到日期由学校当年通知为准。
- 报到后先回宿舍安顿，再听学院安排开学典礼、入学教育。
- 正式上课一般在报到后 1–2 天内展开。

## 通用日程框架

以往年新生手册为例（具体日期以本级学校通知为准）：

| 时间节点 | 内容 |
| --- | --- |
| 报到 D-1、D 天 | 新生集中报到；辅导员在学院报到点等待 |
| 报到当天 | 缴费、领取宿舍钥匙、领取一卡通、领取新生礼包 |
| 报到当天 | 学院见面会：辅导员、班主任、班委见面 |
| 报到后第 1 天 | 学院开学典礼 |
| 报到后第 2 天 | 学校总的开学典礼（多在体育馆或第一田径场） |
| 报到后第 2–3 天 | 正式开始上课 |
| 报到后 1–3 周 | 入学教育、体检、注册、班级活动 |
| 报到后 3–4 周 | 军训（详见 [军训指南](/library/newcomer/军训指南)） |

## 报到当天注意事项

- 听从学院迎新志愿者（辅导员或高年级学长学姐）的引导。
- 报到当天事情多、节奏快，跟着流程走比自己乱翻通知更快。
- 携带的证件、材料清单请见 [开学物品指南](/library/newcomer/开学物品指南)。
- 家庭经济困难同学请直接找绿色通道窗口办理，不需要担心筹措不齐费用。

## 开学典礼

- 学院开学典礼通常在报到后第 1 天。
- 学校总的开学典礼通常在报到后第 2 天上午。
- 学校总开学典礼参与人员一般包括：全体校领导、校党委委员、院士、长江学者、国家级教学名师、校属各单位党政负责人、各学院党委副书记 / 团委书记、本科生 / 研究生辅导员、新生班主任及全体新生。
- 场地一般在体育馆或田径场，容量大但保暖要注意。

## 正式上课

- 通常在报到后 1–2 天开始正式上课。
- 第一次上课前先通过教务系统确认课表、教室、任课老师。
- 走错教室是常见现象，可以提前问问身边的同学或学长学姐。

## 使用小建议

1. 抵达前一晚检查一次证件清单，避免第二天匆忙翻找。
2. 报到当天穿舒适的衣服和运动鞋，行李较重可以提前联系家人协助。
3. 手机保持满电、开机，加辅导员微信 / QQ 后不要错过通知。
4. 涉及具体报到时间、班车安排、接站志愿者点，请以学校当年发布的迎新通知为准。
