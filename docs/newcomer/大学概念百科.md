# 大学概念百科

大学阶段有一批高频出现的术语：**绩点、选修课、综测、保研、选调、国奖、绿色通道** 等等。这份百科按 “学业、升学、身份权益、毕业档案” 四类，把新生最容易困惑的概念一次讲清楚。

## 一、学业核心概念（决定能否毕业、保研、评优）

### 1. 绩点

- 成绩排名的核心依据，分数越高绩点越高。
- 直接决定 **评奖、保研、转专业资格**。

### 2. 选修课

- 学生自主选课的部分。
- 分 **专业选修** 和 **全校公选（通识 / 自主发展）**。
- 修满对应学分即可，学分不足会影响毕业。
- 详见 [选课与通识课](/library/study/选课与通识课)。

### 3. 补考

- 期末挂科后，开学初可 **免费补考一次**。
- 补考不过通常需要重修。
- 详见 [学分、成绩与课程考核](/library/study/学分成绩与课程考核)。

### 4. 综测（综合素质测评）

- 综合考察学业、社会工作、志愿服务、竞赛、文体等表现。
- 综测排名与评奖、保研、评优挂钩。

### 5. 学业预警

- 学期结束后系统对不及格课程数、绩点低于门槛的学生发出预警。
- 学业预警一般会伴随谈话、家长联系、辅导等干预。

## 二、升学路径核心概念

### 1. 推免保研（免考研）

- 通过学院和学校推免评审，无需参加全国研究生入学考试即可读研。
- 详见 [保研与科学竞赛](/library/study/保研与科学竞赛)。

### 2. 单列保研 / 竞赛保研

- 在国家级学科竞赛决赛中获得国家级一等奖及以上的学生，可通过单列指标或竞赛保研通道，由学校推免工作领导小组审批。

### 3. 学硕 / 专硕 / 直博

- **学硕**：学术型硕士，偏研究，学制 2–3 年。
- **专硕**：专业型硕士，偏应用，学制 2–3 年。
- **直博**：本科毕业直接攻读博士，不单独获得硕士学位。

### 4. 选调生

- 各省针对重点高校招录的党政干部，比普通公务员层级更高、晋升更快，是大学生公考最优路径之一。
- 有严格的政治面貌、绩点、综测、无违纪要求。

## 三、奖助与经济资助核心概念

### 1. 国家级

- 国家奖学金：**8000 元 / 生 / 年**（本科），面向二年级以上本科生中特别优秀者。
- 国家励志奖学金：**5000 元 / 生 / 年**（本科），面向二年级以上品学兼优的家庭经济困难学生。

### 2. 国家助学金（纯补贴、不用还）

- 分 **三档：一等 4400 元 / 二等 3300 元 / 三等 2200 元**，平均资助标准 3300 元 / 生 / 年。
- 大一即可申请。

### 3. 绿色通道

- 家庭经济困难新生如暂时筹集不齐学费和住宿费，可先通过绿色通道入学。
- 详见 [奖学金与助学金政策](/library/tools/奖学金与助学金政策)。

## 四、纪律与身份规则

### 1. 考勤旷课制度

- 查课严格，旷课累计达标直接取消课程考试资格、挂科。
- 代签到、虚假签到一律违纪通报。

### 2. 宿舍安全红线

- 严禁大功率电器、明火、私调床位、夜不归宿、外宿不报。
- 违规直接通报扣分，严重者进入违纪档案。

### 3. 处分档案影响

- 所有处分入档案。
- 影响 **评优、入党、保研、考公、就业**。

## 五、学生身份与权益概念

### 1. 学生会

- 主要角色：校院两级学生自治组织，组织活动、传达学生诉求。
- 参与好处：**锻炼能力、综测加分、校级荣誉**。

### 2. 入党

- 大学入党含金量远高于社会入党。
- **大一是最佳申请窗口期**，无违纪、成绩良好为基础条件。

### 3. 兵役 / 参军

- 应征入伍服义务兵役、招收为军士的学生，可享受国家教育资助政策，包括学费补偿、国家助学贷款代偿、学费减免。
- 详见 [奖学金与助学金政策](/library/tools/奖学金与助学金政策)。

## 六、毕业与档案核心概念

### 1. 毕业与学位

- 本科通常 4 年制，湘大最长可读 **6 年**，超期注销学籍。
- 挂科当年通常取消所有奖助、评优、保研资格。
- 作弊、严重学术不端、留校察看处分未解除，不授予学位。

### 2. 学籍档案

- 记录本科阶段成绩、奖惩、组织关系等信息。
- 是升学 / 就业过程中最重要的书面依据。

### 3. 二战 / 档案留校

- 考研二战可申请档案留校保管 **2 年**，不影响后续升学、就业调档。

## 使用小建议

1. 新生第一周把这些概念背熟，比后期一个个查通知效率高很多。
2. 涉及 “到底几分算优秀 / 什么时候申请 / 怎么办手续”，请以本级学校发布的通知为准。
3. 有争议或不清楚的概念，先问辅导员，别在群里传谣。
