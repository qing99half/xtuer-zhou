# xtuer-zhou UI 设计方案 v0.1

## 1. 设计定位

面向湘潭大学新生的实用型校园指南工具。整体风格年轻、清爽、可信，不做夸张营销页，不使用大面积单一紫蓝渐变。

## 2. 视觉原则

- 信息优先：标题、模块、入口清晰可扫读。
- 校园感：使用温和红色、湖蓝、浅米色作为点缀。
- 知识库感：大面积留白、卡片分组、弱阴影。
- 移动端优先：导航、卡片、问答入口在手机上可自然纵向排列。

## 3. 色彩建议

- 主色：`#b42318`，用于重点按钮、当前状态、核心强调。
- 辅色：`#0f766e`，用于可信标签、资料状态。
- 背景：`#fffaf2`、`#f8fafc`。
- 正文：`#1f2937`。
- 次级文字：`#64748b`。
- 边框：`#e2e8f0`。

## 4. 布局结构

### 顶部导航

- 左侧：项目名“湘潭大学新生攻略”。
- 中间 / 右侧：`首页`、`入学指南`、`资料库`。
- 右侧按钮：`加群咨询`，点击打开联系方式弹窗。
- 移动端：导航换行或横向滚动，按钮保持可见。

### 联系方式弹窗

- 标题：`加群咨询 / 联系我们`。
- 内容卡片：微信二维码占位、QQ 新生群、湘大表白墙。
- 明确提示：群聊为民间自建，注意甄别信息。
- 操作：关闭按钮明显，弹窗不遮挡关键状态。

### 首页

1. Hero 区：一句定位 + AI 问答输入占位。
2. 新生入学路线：4 张模块卡片，报到流程、宿舍生活、军训指南、学习入门。
3. 咨询入口：QQ 新生群、湘大表白墙二次强调。
4. 资料状态提醒：所有正式资料需审核后展示。

### 入学指南页

- 统一模板卡片：模块简介、核心问题、指南内容占位、注意事项、常见问答、引用来源。
- 4 个模块采用同样结构，便于后续写入内容。

### 资料库页

- 分类导航沿用 `docs/` 目录。
- 当前只展示已审核资料；未审核资料不进入正式列表。
- 空状态明确提示：资料正在整理，待审核后上线。

## 5. 组件规则

- 卡片圆角不超过 `8px`。
- 阴影轻量：只用于主要容器。
- 按钮文案直接，避免花哨。
- 状态标签统一使用小胶囊样式。

## 6. 验收标准

- 首页、入学指南、资料库三条路径清楚。
- 联系方式入口在导航和首页都能看到。
- 移动端不出现明显横向溢出。
- 未审核资料不会被包装成正式资料。
