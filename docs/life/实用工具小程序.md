# 实用工具小程序

湘大新生入学后，除了官方门户和 App，还有一批 **微信小程序** 是日常生活的高频入口。这份指南把最常用的几个先列出来，方便对号入座。

## 一句话理解

- **微生活 / 校园集市**：查课表、看校园通知、了解校园活动。
- **五维时空**：报名参加各类校园活动。
- **智慧湘大**：水电缴费、宿舍报修。
- **爽提**：校园云餐厅点餐配送（详见 [学校里的美食家](/library/life/学校里的美食家) 和 [快递收发驿站通](/library/life/快递收发驿站通)）。

## 常用小程序简介

### 湘潭大学微生活 / 校园集市

- 主要功能：**随时查看课表、发帖求助、了解校园趣事**。
- 常用场景：跑教室前查课表、遇到问题问其他同学。

### 五维时空

- 主要功能：**报名参加各类活动**。
- 常用场景：讲座、社团招新、志愿活动、比赛报名。

### 智慧湘大

- 主要功能：**水电缴费、宿舍报修**。
- 常用场景：查看宿舍电量、缴纳生活费、报修水管 / 灯管 / 门锁问题。

## 其他值得关注的入口

- **信息门户**（`i.xtu.edu.cn`）：教务、缴费、通知等常用系统的统一入口。
- **教务系统**：查课表、成绩、选课，详见 [湘大教务系统使用入门](/library/tools/湘大教务系统使用入门)。
- **图书馆微信公众号**：座位预约、开馆时间、活动报名。
- **辅导员 / 班主任微信 / QQ 群**：通知转发和答疑最快的渠道。
- **湘大表白墙**：学生自发的社交入口，可关注热点、寻物启事、交流咨询。

## 使用小建议

1. 报到当天就把上述小程序全部关注 / 添加一遍，节省后续查找时间。
2. 缴费类小程序注意 **官方来源**：只走信息门户 / 智慧湘大等官方入口，不要在其他渠道输入密码。
3. 报名类小程序（如五维时空）通常有报名截止时间，别错过。
4. 如果小程序无法访问，先看是否是校园网 / VPN 问题；再看是否是系统维护。
5. 涉及 “今年官方小程序名单、最新版本、功能更新” 请以学校信息化建设与管理处或学生工作部通知为准。
