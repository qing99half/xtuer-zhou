# 班委职务与竞选

大学的班委和高中不一样，责任更清晰、能力锻炼更足。这份指南把 **常见班委职务的工作内容 + 详细职责 + 面向新生的建议** 讲清楚，方便你在班委选举时有的放矢。

## 一句话理解

- 常设班委：**班长、副班长、团支书、学习委员、组宣委员、生活心理委员、权益委员** 等。
- 每个班委既是 **工作岗位** 也是 **能力锻炼平台**，社工经历可用于综测加分、评优评先。
- 竞选看的是 **有没有心、有没有热情**，不是 “此刻拥有多少经验”。

## 一、简要职务清单

### 班长

- 主持全班各种工作有序推进，对班级工作全面负责，统筹安排班内事务。
- 及时传达工作任务，向辅导员请示工作。
- 定期参加班团会议。
- 按时召开班委会、班会；定期总结班委会工作，研究班内情况，布置之后方针。
- 配合和协调班级成员工作。

### 团支书

- 管理同学们在团内的档案。
- 上传同学们的资料。
- 开展每月一次的团日活动。
- 管理志愿服务等。
- 协助班长工作。
- 定期参与班团会议。

### 学习委员

- 每学期开学分发教材。
- 收发部分科目作业。
- 及时向同学们发布竞赛、SIT 项目等信息和资讯，帮助同学们做生涯规划。
- 关注班级同学学习情况，定期组织自习、讲题等活动。
- 协助班长、团支书工作。

### 权益委员

- 每天查寝填表。
- 转发相关通知。
- 收班费和教材费。
- 收集贫困生的表。
- 协助班长、团支书工作。

### 心理委员

- 定期参与心理委员会议。
- 担任朋辈互助员，加入朋辈互助员群，一学期负责开展一次心理团辅。
- 加入心理委员群，一些心理活动和心理相关的通知需要向班群里转发。
- 协助班长、团支书工作。

## 二、详细介绍

### 班长（负责班级全面工作）

- 落实班级活动，联合班委开展各项工作，筹备组织班会及其他重大班级活动。
- 积极配合班主任进行班级教学及管理工作，反映同学们的意见和需求。
- 强化班级纪律，保证教学秩序，树立良好班风，维护集体荣誉。
- 支持并督促其他班委工作开展情况。

### 团支书（负责班级思想、党团方面工作）

- 负责班级同学的政治学习，了解思想动态。
- 积极开展团员教育工作，包括团课、团日活动、青年大学习等。
- 完成班级同学的团员、党员发展工作。
- 入党、入团工作流程的学习与传达。
- 班级成员的团费收缴。
- 帮助配合其他班委的工作。

### 学委（负责班级学习方面工作）

- 及时传达学校学院关于教学教务方面的通知。
- 安排各门课程作业的收发。
- 负责组建或组织加入学科课程群，及时将各门课程作业考试等信息下达给同学。
- 因 “班” 制宜思考调动班级学习气氛的方法。

### 副班长（负责协助其他班委开展工作）

- 协助班长进行各项工作开展。
- 进行班级工作总结、统计班级活动情况。

### 组宣委员（负责班级活动组织及宣传方面工作）

- 参与协助班级重要活动组织策划，增强班级凝聚力。
- 班级活动过程中拍照及活动宣传稿件撰写。
- 积极动员同学参加院、校各项活动。

### 生活心理委员（负责班级心理健康及资助管理方面工作）

- 负责班费收取管理，班级活动开支等。
- 及时传达学校、学院关于资助方面的通知，协助辅导员完成班级奖助学金发放。
- 组织同学做好班级后勤保障工作。
- 积极宣传心理健康知识，组织开展心理健康相关活动。
- 对心理有困惑的同学，主动进行沟通，耐心开导。

## 三、竞选与新生建议

- 参与竞选，意味着你愿意承担 **一份责任**：无论是学习上的互帮互助，还是活动中的精心策划，你的每一份付出都会化作班级前进的动力。
- 不用担心自己不够优秀——**每一个伟大的开始都源自于一次勇敢的尝试**。
- 在班委工作中，你会锻炼 **团队协作、领导力、组织策划、公文写作、沟通协调** 等能力。
- 综测评分中，班委工作通常有加分，参与班委也是入党、评优的重要参考。
- 竞选时可以从 “**为什么参选 → 优势 → 打算做什么 → 面临压力时的对策**” 四个层面准备。

## 四、常见问题

### Q1. 大一新生能担任班长吗？

- 可以。班委由全班同学在班主任 / 辅导员组织下选举产生，一年级同学同样能参选。

### Q2. 担任班委会不会耽误学习？

- 时间管理是关键。多数班委每周投入约 2–5 小时，认真规划可以兼顾学习。
- 与辅导员和同学建立良好沟通，工作会更顺利。

### Q3. 一学期担任一个班委，还能加入学生会 / 社团吗？

- 完全可以。班委 + 学生会 / 社团 = 多渠道锻炼。
- 注意工作量不要超出个人承受。

### Q4. 换届怎么办？

- 多数班级每学年或每学期换届一次，具体以班主任 / 辅导员安排为准。

## 使用小建议

1. 新生第一次班会前想清楚 “自己想承担哪块工作”，避免临场压力大。
2. 如果没竞选上班委，也可以担任 **课代表、寝室长、活动负责人** 等岗位，同样能锻炼能力。
3. 综测加分的说明可以在学工部或学院学工办查询。
4. 涉及具体班委名单、每年调整，请以你所在班级的班会决议为准。
