一、入学前线上预约办理（优先推荐，省时避坑）
1. 正规预约渠道（仅认可以下渠道）
湖南电信校园官方公众号、电信官方校园专属客服、学校官方新生群发布的电信正规预约二维码。坚决拒绝陌生私人链接、非官方朋友圈海报、个人私发预约二维码，这类渠道多为外包代理，套餐虚标、扣费乱象频发，售后无法保障。
2. 完整线上预约操作步骤
① 进入官方预约页面，院校精准选择湘潭大学，如实填写本人姓名、身份证号码、录取专业、预计报到时间、意向宿舍园区（琴湖/金翰林/西湖/兴湘）；
② 仔细浏览在售两大套餐：优惠锁定套餐、无合约灵活套餐，逐条核对月租金额、通用流量、校园流量、通话时长、宽带带宽、流量限速标准、注销规则，选定合适套餐后提交预约；
③ 系统完成信息登记并生成预约凭证，全程无需线上提前缴纳任何费用，仅留存预约截图、客服聊天记录作为凭证即可。
3. 报到线下领卡激活流程
① 报到当天携带本人身份证原件，前往电信在宿舍园区设立的官方临时服务站；
② 工作人员核验预约信息与身份证，发放电话卡，完成人脸实名认证开卡，预缴首月话费激活手机号；
③ 登记准确宿舍信息，后台开通宿舍宽带使用权限；
④ 当场连接宿舍网络，使用手机、电脑分别测试网速、延迟、多设备登录情况，确认套餐与预约描述完全一致，妥善保管纸质业务回执单、缴费票据，方便后续售后维权。
4. 线上预约核心优势
提前锁定开学限量学生优惠套餐，不用在报到高峰期排队等候；线上可提前把资费、套餐类型、注销、售后全部疑问沟通清楚，现场仅需领卡激活，极大节省报到时间，避免现场工作人员盲目推销高价套餐。
二、开学线下现场直接办理全流程
1. 唯一可信线下正规办理点位
（1）长期固定点位：南苑商业街电信校内直营营业厅，全年工作日正常营业，可随时办卡、改套餐、账单核对、故障报修、号码注销；
（2）短期临时点位：新生报到3–5天内，琴湖、金翰林、西湖、兴湘四大宿舍园区楼下统一设置电信官方临时驻点，配备标准化工作台、电信正式工牌、纸质套餐价目公示表，两类套餐清晰公示。
2. 绝对需要拒绝的办理场景
任何以学长学姐身份直接进入寝室上门推销、无固定摊位、仅口头介绍套餐、拒绝出具纸质协议的人员，全部为校外外包代理，售后无保障，一律直接回绝。
3. 详细线下办理完整步骤
① 多方对比筛选：仔细查看电信两大套餐细则，把月租、流量构成、通话时长、宽带规格、套餐类型（锁定优惠/无合约）、注销条件逐条确认，切勿当场冲动开卡；
② 实名身份核验：必须本人持身份证原件到场做人脸实名认证，严禁借用、冒用他人身份证代办，否则后续宽带绑定、账号售后都会出现无法解决的问题；
③ 细读纸质服务协议：明确所选套餐是优惠锁定还是无合约版本，确认流量达量限速阈值与限速后网速、有无默认开通付费增值业务；
④ 话费充值开卡：缴纳首月话费激活手机卡，工作人员录入宿舍信息开通宽带权限；
⑤ 现场网络实测：当场完成联网测试，核对全部权益无误后收好所有纸质单据。
三、办卡必核实5项关键信息（避坑核心）
1. 明确套餐类型：是优惠周期锁定套餐，还是可随时注销的无合约套餐，清楚两种模式各自规则；
2. 通用流量、校园专属流量额度，流量封顶限速标准以及限速后实际网速；
3. 宿舍宽带为独享带宽，账号最多支持同时在线设备数量；
4. 无合约套餐是否支持电信APP、小程序线上直接注销；优惠锁定套餐优惠周期结束后如何自主更换套餐、销户；
5. 套餐内是否自带付费增值业务，是否可以一键全部关闭。
四、售后、销户、特殊情况处理细则
1. 话费流量扣费异常、套餐纠纷
优先联系办卡点位电信校内专属客户经理，留存联系方式；协商无果携带回执单据前往南苑电信直营营业厅，现场核对账单，取消不合理扣费业务。
2. 无合约套餐注销
使用期间任意时间，可通过中国电信APP线上提交注销，结清话费即可销户，剩余余额可退还，全程无违约金、无任何附加费用。
3. 优惠锁定套餐到期后续处理
优惠周期结束，套餐自动转为社会通用套餐，学生拥有三种自主选择：①保留号码更换电信任意普通套餐继续使用；②办理携号转网更换其他运营商；③线上APP直接结清话费注销号码，无任何额外费用。
4. 在校特殊情况提前解约（优惠锁定套餐）
因参军、休学、退学等学校认可的特殊情况，携带学校开具证明、身份证前往校内电信营业厅，可申请免除相关约束正常注销；办卡前可提前知悉相关条款。
5. 手机卡丢失、损坏补卡
本人携带身份证前往南苑电信营业厅即可当场补卡，电信校园卡每年享有1次免费补卡权益。
五、科学办理时间规划
1. 报到前3–7天：线上了解电信两款套餐，选定意向类型完成预约，留存全部沟通凭证；
2. 报到当日：线下领卡实名激活，开通宽带并完成全设备网速测试，收好所有纸质单据；
3. 报到1–3天试用观察：正常使用3天，核查流量到账、通话信号、宽带稳定性，出现问题第一时间联系工作人员调整；无合约套餐试用不满意可直接线上注销；
4. 入学1个月最终确认：关闭所有不必要增值业务，确认套餐完全适配个人使用需求，即可稳定使用。
六、高频常见问题解答
1. 只用教学楼、图书馆XTU-FREE免费WiFi可以吗？
XTU-FREE为全校公共共享免费网络，仅支持临时查资料、浏览网页，高峰期大量学生同时接入会出现严重卡顿、掉线，无法支撑电脑网课、大型软件下载、网络游戏，只能作为应急使用，不能替代宿舍独立宽带。
2. 寒暑假离校，校园流量还能用吗？
校园专属流量仅限湘潭大学校园全域范围，寒暑假回家只能使用全国通用流量，通用流量耗尽将按照套餐标准资费计费，离校前可按需叠加电信全国短期流量包。
3. 可以多人共用一张电话卡绑定宽带吗？
不可以。湘大校园网后台具备严格的账号实名与设备监测机制，共用账号会直接封禁宽带，需本人前往电信营业厅申请解封，情节严重会永久收回宽带使用权限。
4. 无合约套餐后续还能换回优惠锁定套餐吗？
在校学籍有效期内，可随时前往电信校内营业厅，申请切换为优惠锁定套餐，享受长期稳定低价。
