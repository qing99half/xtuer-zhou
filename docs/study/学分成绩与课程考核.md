# 学分、成绩与课程考核

学分、绩点、考核方式是本科阶段最核心的评价指标，它们影响毕业、学位、保研、奖学金、评优等几乎所有关键结果。这份指南把常见问题按 “**学分怎么算、成绩怎么算、考核怎么做**” 三个层面梳理。

## 一句话理解

- **学分**：完成一门课程规定学时，就获得对应学分。
- **成绩**：包含平时、期中、期末、实验、报告等多种权重，最终由任课老师登录教务系统。
- **考核**：分为考试课和考查课，评价方式不同，绩点换算也不同。

## 一、学分

- **总学分要求**：每个专业培养方案给出毕业总学分下限（例如经济学专业本科培养所修学分原则上不少于 117 学分）。
- **学分结构**：一般包括——
  - 公共基础课（含 A 类 / B 类课程）
  - 专业基础课
  - 专业主干课
  - 专业选修课
  - 通识 / 自主发展课程
  - 实践 / 实验 / 毕业设计
- **通识 / 自主发展课程**：培养方案会给出学分下限，达不到会影响毕业。
- **重修 / 补修**：未通过课程要参加 **补考** 或 **重修**，学分要重新修够，成绩以最终一次登记为准（具体以教务规定为准）。

## 二、成绩

### 成绩构成

- 每门课通常包含 **平时 + 期中 + 期末**，具体权重由任课教师在课程大纲中说明。
- 实验课、设计类课程可能会包含实验报告、答辩、作品评分等。
- 部分课程使用 **考查制**，最终成绩以“优秀 / 良好 / 中等 / 及格 / 不及格”呈现。

### 常见成绩类型换算（举例）

以数学与计算科学学院公开细则为例，考查课程成绩换算参考：

- 优秀 → 92.5 分
- 良好 → 80 分
- 中等 → 69.5 分
- 及格 → 62 分
- 不及格 → 30 分

考试课程成绩以学生 **第一次参加课程考试的成绩** 计算。

> 上述换算主要用于加权 GPA / 推免评价，不同学院可能采用不同换算表，具体以本学院细则为准。

### 绩点（GPA）

- 加权平均成绩 = Σ（课程成绩 × 学分）/ Σ 学分
- 绩点算法在培养方案或教务处“教学管理”栏目会有说明。
- 保研、奖学金、评优等场景通常看 **必修 + 专业选修课的加权平均成绩** 或 **绩点**。

### 成绩公示与复核

- 成绩由任课教师登录教务系统；学生在教务系统查看。
- 如认为成绩明显有误，可申请 **成绩复核**，具体窗口以教务处通知为准，逾期不受理。

## 三、课程考核

### 考试课

- 期末统一安排考试时间、考场、监考。
- 缺考 / 违纪按学校教务规定处理，情节严重的进入违纪档案。
- 考试作弊、抄袭、代考等属于严重违纪，学校对学位授予有明确红线。

### 考查课

- 一般以论文、报告、作品、平时表现等作为考核依据。
- 考查制成绩不代表容易通过；不合格同样需要补考或重修。

### 实验 / 实践课

- 有出勤、报告、操作、答辩等多种考核环节。
- 缺席实验课通常直接影响成绩，需要及时请假补做。

### 毕业设计 / 毕业论文

- 大四集中期，由指导教师、系 / 学院答辩委员会共同评价。
- 学位授予的核心考核环节之一，务必按时提交、按格式规范撰写、按学院要求答辩。

## 常见误区

- ❌ “考试不来就是零分”：缺考按学校规定处理，除了零分还有可能影响学籍、评奖资格。
- ❌ “补考通过 = 首次通过”：多数推免细则中，首次考试不及格即视为不满足推免资格。
- ❌ “考查课好过”：考查课同样有不及格风险，尤其是报告类课程要按时按质提交。
- ❌ “大一学分不重要”：大一是绩点基础年，多数保研评价看的是前 3 年累计成绩。
- ❌ “成绩会自动同步显示”：任课老师有登录截止时间，学期末成绩会逐步刷新，不会同时全部出现。

## 使用小建议

1. 学期初就把每门课的 **课程大纲（含考核比例）** 拿到手，做好时间规划。
2. 平时分要重视：出勤、作业、小组任务、课堂表现都可能占 30–50%。
3. 期末周关注 **考试时间冲突** 和 **教室变动**，任何异常尽早找辅导员和教务办反馈。
4. 成绩出来后及时截图存档；如需成绩单盖章，走教务处流程。
5. 涉及 “重修如何选课、绩点如何加权、补考安排、学分置换” 等具体规则，一律以教务处官方文件和你所在学院通知为准。
