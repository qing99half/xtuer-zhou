# 选课、修课与访学

“选课、修课、访学”这三件事对本科同学来说，是每学期都会碰到的高频动作。前面单独整理过 [选课与通识课](/library/study/选课与通识课) 讲通识 / 自主发展课程的选课细节；这一篇作为总览，把 **修课过程中的关键动作** 和 **访学 / 联合培养的学分认定** 一起说清楚。

## 一句话理解

- 选课是入口，修课是过程，访学是校外场景下的“修课”。
- 三者共同点：都会形成 **学分认定** 结果，最后写进你的成绩单。
- 三者都要按 **教务处 + 学院 + 国际交流处** 发布的通知走，不可以自作主张。

## 一、选课要点

- 每学期教务处发布《关于做好 XX 学期自主发展课程选课通知》和《湘潭大学教务系统选课操作手册》。
- 常见附件文件名示例：
  - `2026-2027-1学期自主发展课程选课通知.doc`
  - `湘潭大学自主发展课程选课操作手册.docx`
- 具体细节详见 [选课与通识课](/library/study/选课与通识课)。

## 二、修课要点

### 修课过程的关键动作

- **课程签到 / 出勤**：多数课程有出勤要求，缺勤会影响平时分和学位相关评价。
- **作业 / 报告 / 实验**：按课程大纲提交，注意时间节点。
- **课程调整**：如发现所选课程明显不合适，可以在 **退选窗口** 内退课，逾期不受理。
- **考试 / 考核**：见 [学分、成绩与课程考核](/library/study/学分成绩与课程考核)。

### 免修 / 学分替代 / 学分认定

- 部分课程可以通过考试、竞赛、社会实践、跨校修读等方式申请 **学分认定 / 学分替代**。
- 具体规则以教务处、学院教务办的通知为准，通常要提交申请表 + 佐证材料。
- 已在其他学校 / 项目修过的课程，需要走 “**学分认定**” 流程转换为湘大培养方案的对应课程。

## 三、访学与联合培养

湘大鼓励学生在保证学业质量的前提下，通过 **访学 / 联合培养 / 交换项目** 拓展视野。

### 常见路径

- **国内交换**：与国内其他高校签订的交换协议，短期或一学期学习。
- **国际交流 / 联合培养**：由国际交流处协调的境外院校项目，短期访学、暑期学校、长期联合培养等。
- **企业联合培养 / 校企共建实验班**：例如化学学院与中国科学院化学研究所、株洲时代新材料的“3+1”联合培养模式，材料类师昌绪班、韶峰科创班等实验班的校企共建方向。
- **微专业 / 拓展课程**：属于跨专业选课，但通常仍在校内完成，不属于严格意义上的访学。

### 学分认定流程（通用）

1. **申请前**：向所在学院和目标院校 / 项目方确认课程清单、学分对照关系。
2. **申请**：填写 **境外 / 校外修读课程学分认定申请表**，附课程大纲、成绩证明。
3. **审核**：由所在学院学位委员会或教学委员会审核。
4. **教务处备案**：审核通过后由教务处备案，学分录入教务系统。
5. **注意事项**：
   - 认定通过的课程学分会计入总学分，但绩点计算规则以教务处规定为准。
   - 部分核心课程可能 **只认定学分、不替代培养方案课程**，仍需要按培养方案修满对应课程。

### 咨询与申请入口

- **国际交流处**：负责境外访学、暑期学校、联合培养项目的对接。
- **教务处**：负责学分认定、成绩录入、注册管理。
- **所在学院 / 项目方**：项目名单、面试、行前培训等具体流程。
- **信息门户 / 教务系统**：查看和提交部分线上审批流程。

## 常见误区

- ❌ “选修课随便退”：退选窗口有限，逾期无法退课。
- ❌ “没参加就是没选”：只要提交了选课订单，即使不去上课也算注册；建议不选就不选，选了就要跟到底。
- ❌ “校外修课自然会转学分”：必须提前走学分认定申请，事后没法补办。
- ❌ “访学一定能替代培养方案核心课”：部分核心课需要按培养方案在校内修读，认定时会区别处理。
- ❌ “微专业 = 辅修”：微专业和辅修性质、学分要求、学位授予都不同，不要混淆。

## 使用小建议

1. 学期初就在教务处官网收藏 **选课通知发布栏目**，减少错过窗口的概率。
2. 想申请访学的同学，提前 1–2 个学期开始准备语言、成绩、申请材料。
3. 学分认定要 **先审批、后修课**，不要“先斩后奏”。
4. 访学期间保持和辅导员、学院教务办的沟通，避免出现学籍 / 学分处理问题。
5. 涉及具体项目（如具体交换校名、招募时间、名额）请以本学期国际交流处和学院通知为准。
