# 校园文化

湘大的校园文化底色是 “**毛主席亲自倡办 + 综合性大学 + 湖湘文化**”。校徽、校训、校歌是最常见的三张“文化名片”，也是入学教育、演讲、社团宣讲反复出现的知识点。

## 校徽

- 圆形设计。
- 中英文校名分居上下两侧。
- 中文校名 “湘潭大学” 为 **毛体字**，由毛泽东同志亲笔题写。
- 校徽整体传递出学校的历史渊源与综合性大学的定位。

（校徽实物图片需以官方素材为准，请查看官网 `www.xtu.edu.cn` 或迎新材料。）

## 校训

**博学笃行，盛德日新**。

- “博学笃行”出自《礼记·中庸》：博学之，审问之，慎思之，明辨之，笃行之。
- “盛德日新”出自《周易·系辞》：日新之谓盛德。
- 校训精神强调既要广博学习、扎实实践，又要在品德修养上不断更新、追求卓越。

## 校歌

湘大校歌引用了 **毛泽东同志《沁园春·长沙》** 中的诗句，与学校历史渊源呼应。

- 歌词、词曲作者、演唱时机（校庆、开学典礼等）以官方素材为准。
- 校歌音频文件请到官网、学校融媒体中心或迎新平台获取。

## 校园文化活动线索

湘大常见的校园文化活动包括但不限于：

- 校庆日相关活动（9 月为学校创办纪念月）。
- 每年迎新季、毕业季组织的文化演出、宣讲活动。
- 学生会、学生社团联合会、大学生艺术团、��究生会主导的文体活动。
- 各学院组织的院庆、学科日、学术周。

具体活动清单、举办频率、参与方式，请见对应学生组织和学院的通知。

## 面向新生的实用建议

1. 报到时留意校徽、校训在校园各处的呈现（教学楼、图书馆、纪念碑等）。
2. 参加校史馆参观是了解校园文化最直接的方式，通常在入学教育或军训期间安排。
3. 校歌、校训是新生入学教育考核、社团面试常见问题，可以提前熟悉。
4. 想深入了解校园文化，可以关注学校党委宣传部、融媒体中心、学生工作部的推送。
