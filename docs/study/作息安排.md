# 作息安排

湘大新生刚入校最想弄清楚的三件事：**几点上课、几点吃饭、几点熄灯**。这份指南先给出通用节奏，再指向权威入口，方便你按学期查最新版本。

## 一句话理解

- 学期作息通常按 **早课 → 上午 4 节 → 午休 → 下午 4 节 → 晚自习** 的框架安排。
- 军训期间、正式教学期、期末周、寒暑假前后的作息可能会切换。
- 一切以 **本学期教务处发布的作息表 / 校历** 为准，不要用学长学姐旧版直接对号。

## 通用作息框架（示意）

以下是国内大多数高校常见的作息节奏，湘大具体时间以当学期校历为准：

| 时段 | 内容 |
| --- | --- |
| 6:30 – 7:30 | 早读 / 早锻炼 / 早餐（因专业和学期不同） |
| 8:00 – 12:00 | 上午上课，通常 4 节课 |
| 12:00 – 14:00 | 午休 + 午餐 |
| 14:30 – 17:30 | 下午上课，通常 4 节课 |
| 17:30 – 19:00 | 晚餐 |
| 19:00 – 21:30 | 晚自习 / 晚课 / 学生活动 |
| 22:30 – 23:30 | 宿舍熄灯（片区不同时间略有差异） |

⚠️ 这个表只是通用框架，不是湘大官方作息表。具体到多少分开始、每节课多长、什么时候大课间，都以教务处当学期通知为准。

## 军训期间

- 军训作息通常以 **早集合 – 训练 – 午休 – 训练 – 晚集合 / 讲评** 为主。
- 大概率会比日常上课更严格，早集合可能在 6:30 前。
- 请以军训带训教官、辅导员当日通知为准；淋雨、酷暑、身体不适请及时报告。

## 期末与考试周

- 期末周教室、图书馆、自习室使用高峰。
- 考试时间由教务处统一发布，一般集中在学期最后 1–2 周。
- 期末公共选修课调课、补课通知会集中发布，务必留意教务系统和辅导员群。

## 假期作息

- 寒暑假前通常有 “**离校时间通知 + 学生离校登记 + 宿舍关闭时间**” 三份文件。
- 假期期间图书馆、部分食堂开放时间会缩短。
- 校园卡余额、社保、快递等假期前建议自查一遍。

## 查最新作息表的入口

- **教务处官网**：`https://jwc.xtu.edu.cn`
- **教务系统**：登录后能看到本学期课表、教室、任课教师、考试安排。
- **辅导员 / 学院教务办**：他们最先拿到调课、放假、军训作息等通知。
- **迎新材料**：报到时会发放本学期校历 / 作息表纸质版或电子版。

## 常见误区

- ❌ “学长学姐的作息表能直接用”：每学期、每级、每个专业课表可能不同。
- ❌ “有课表就够了”：课表只显示上课时间，早锻炼、晚自习、集会等要看年级通知。
- ❌ “熄灯 = 断电”：不同宿舍片区熄灯规则不同，个别片区仍能给风扇 / 台灯保留一定电量，以宿舍管理规定为准。

## 使用小建议

1. 军训期间尽量提前 15 分钟到集合点，避免因为片区距离迟到。
2. 大一大二尽量把 **早课全勤** 作为基本习惯，能省下很多补课时间。
3. 涉及具体节次时间（如“第 3 节 10 点几分下课”）请每学期从教务系统再核对一遍。
4. 遇到调课、停课通知，先在教务系统查证，再问辅导员或教务办确认。
